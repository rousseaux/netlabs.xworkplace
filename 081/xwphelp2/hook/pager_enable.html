The "Enable" checkbox determines whether &xwp; allows you to use virtual desktops at all.

If the box is checked, the rest of the page (and the following pages) are activated, and the &pgr; window appears on your screen. You can then switch between multiple desktops, as described in the &pgr; introduction.

If the box is unchecked, &pgr; is deactivated, and all windows that might currently reside on other desktops are brought back to your screen. You then only have one desktop again.

Note: The checkbox can only be checked at all if the &xwp; hook is enabled. If the checkbox is grayed out, go to the "Features" page of the "&xwp; Setup" object and enable the &xwp; hook.
