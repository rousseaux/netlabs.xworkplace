&pgr; supports quite a number of mouse actions.

If &pgr; mouse actions have been enabled on the first &pgr; page in the "Screen" notebook, the following mouse actions are available with the mini windows in the pager:

  • Clicking with mouse button 1 on a mini-window window will bring the corresponding PM window to the foreground and switch virtual desktops, if necessary.

  • Clicking with mouse button 2 will lower the corresponding window to the background (i.e. make it the bottommost of all windows) and switch virtual desktops, if necessary.

  • You can also drag the mini windows with mouse button 2 within the &pgr; window. Once you release mouse button 2, the "real" window will be moved to the desired location. This even allows you to move windows across several virtual desktops, which can come in quite handy.

    If you hold down the Shift key while dragging, the window will have the exact same position on the target desktop as it had on the source desktop. In other words, with the Shift key down, the window is dragged in units of full desktops, not pixels.

The following mouse actions are available with empty areas in the pager (that are not covered by the mini windows):
  • Clicking with mouse button 1 onto an empty area in the &pgr; window will switch to the corresponding desktop without changing the currently active window.

  • Clicking with mouse button 2 onto an empty area in the &pgr; window will bring up the &pgr; &popmenu;.

  • If you hold down the Shift key while clicking with mouse button 2 onto an empty area, the pager window will hide immediately.

  • If you start dragging from empty area, you can move the entire pager window.
