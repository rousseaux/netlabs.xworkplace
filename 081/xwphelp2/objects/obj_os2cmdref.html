The &os2; Command Reference is an online book that documents all commands understood by the operating system. This includes
  • commands that can be specified in the &cfgsys; file;
  • commands that can be used at an OS/2 or DOS command prompt;
  • commands that can be used in batch or command files.
Double-click on the icon to have the online book displayed.

  • Batch and command files
