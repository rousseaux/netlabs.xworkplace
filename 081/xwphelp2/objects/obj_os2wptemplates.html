On the &link_desktop;, In the "&sysfdr;" folder, &os2; has created a folder called "Templates" containing a number of special objects that you can use to create new objects on your system.

These objects are called templates and are used exclusively for creating more objects. The "Templates" folder initially holds those templates that were created by the system, for example, for creating new &link_folders;, &link_progobjs;, and &link_datafiles;.

To create a new object from a template, double-click on the template. Alternatively, you can drag a new object off the template and drop it into another folder. In both cases, a new object will be created that has the same properties as the template (except that it is not a template itself).
