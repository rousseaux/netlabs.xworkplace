The "Locale" object allows you to change country-specific settings for your system.

This folder resides in the &link_syssetupfdr;. Open the Desktop's &popmenu; and select "&syssetup;" to have that folder opened quickly.

A locale is a set of values that define how number, date, time, monetary, and other country-specific values are displayed. Well-programmed applications should be aware of these specifics and respect the selections you make here.

If you mark a locale object as the "default" locale, all compliant applications on your system will use these settings. For example, the &xcenter; date and time widgets will.
