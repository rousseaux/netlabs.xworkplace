XWPTrashCan is the WPS class for the &xwp; trash can object (see WPS classes for details). Since the trash can is just a special folder, in addition to the XWPTrashCan strings listed below, the trash can also supports the strings of the WPObject and WPFolder classes.

To get the trash can object, invoke these strings on the object with the ID <XWP_TRASHCAN>.

EMPTYTRASH=IMMEDIATE

Empties the trash can. No confirmation is displayed.
This is not really a "setting", but rather a command to be performed on the trash can Note that the call does not return until the trash can has been emptied.
