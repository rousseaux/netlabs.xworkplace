Every object has a title associated with it. There are several ways to change an object's title:
  • You can open the object's properties notebook and change the title on the "Icon" page.

  • You can click on the object with the "Alt" key held down. This allows you to edit the title directly in the folder window.
Remarks:
  1. Some objects may refuse to be renamed if the object's title is critical to its operation. You may also not be able to rename an object if it is on a remote network drive to which you do not have write access.

  2. If you change the title of a shadow, the original object will also be renamed.
