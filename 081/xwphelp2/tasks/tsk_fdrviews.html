&xwp; allows you to modify the way folders are displayed on your system in great detail. In general, you can modify view settings in the following places:
  • The &link_wpsobj; has many options which will affect view settings globally, i.e. for all objects on the system.

  • By contrast, you can modify the settings for an individual folder in its own properties notebook.
Among others, you can change the following settings:
  • For each folder, you can select an individual background bitmap on the "Background" page its properties notebook.

    Open the &link_lookfdr; and select the "Scheme palette" to change folder backgrounds globally for all folders on the system.

  • You can change the default view of each folder to either Icon, Tree, or Details view on the "Menu" page in a folder's properties notebook. To change this globally, use the "Folder views" page in the "Workplace Shell" object.

  • You can influence what items are displayed in each of the Icon, Tree, or Details views on the "View" pages in each folder's properties notebook.

  • You can disable the folder menu bar or the status bar either in the "View" menu, or on the "Menu" and "View" pages in the folder properties, or globally on the "Menu" and "Status bar" pages in the "Workplace Shell" object. Note also that you can drag and drop fonts and colors onto any status bar.

  • You can disable many menu items on the "Menu" pages in the "Workplace Shell" object, if you find all the options confusing.
