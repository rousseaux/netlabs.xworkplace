Select this menu item to have &xwp; copy the filename of an object into the &os2; clipboard.

If you hold down the "Shift" key while selecting this menu item, &xwp; will copy the full path also.

If several objects are selected in the folder, all the selected files' names will be copied, separated by spaces.

By default, &xwp; defines the Ctrl+Insert folder hotkey for copying file names without the full path.

You may remove this menu item on the "Menu" pages in the "Workplace Shell" object.
