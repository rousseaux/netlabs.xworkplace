The "Move" menu item will move the selected objects to a different location.

A window will open that allows you to specify the folder where the objects should be moved to. The window has several tabs to specify the target folder in various ways.

Remarks:

  1. If the target folder is already open, you can move the objects more quickly by simply dragging and dropping them onto the target folder. Hold down the Shift key while dragging to make sure the objects are not copied (instead of moved).

  2. Moving a shadow does not move the original object, but the shadow only.

  3. If you are moving files between different drives, be aware that there might be file-system restrictions. See "Restrictions when copying and moving files" for details.
