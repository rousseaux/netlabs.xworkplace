This dialog allows you to view all the WPS classes which &xwp; consists of and whether they are currently registered with the WPS class list.

See "WPS classes" for an introduction to WPS classes.

To register or deregister one of the classes, check or uncheck the respective checkbox and press OK. This will require a Desktop restart.

The class checkboxes have fly-over help associated with them. To find out more about a certain class, put the mouse pointer over a class name and wait a couple of seconds.
