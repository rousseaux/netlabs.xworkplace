On this notebook page you can manipulate some of &xwp;'s internals in case of trouble. Most of the changes will take effect immediately.

Press "Undo" to restore the settings that were in use before you opened this notebook page.

Press "Default" to restore the settings to the installation state.

  • Disable all folder subclassing
  • Disable undocumented menu functions
  • Use "8.Helv" font
  • Disable exception beeps
  • Worker thread priority
  • Beep on priority change
