If "Reboot to..." is selected, the &xshutdown; confirmation window is enlarged to show you the options that you can reboot to. After selecting "Reboot to...", select one of the options on the right.

After pressing "Shutdown", &xwp; will start the shutdown procedure and then run the reboot option that you have specified.

This item is disabled if you have not configured any reboot options on the "&xshutdown;" page in the Desktop's &popmenu; yet.
