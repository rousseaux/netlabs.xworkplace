&os2; allows you to drag and drop fonts and colors on windows to make them change their appearance. Many applications will properly react to those changes and save them internally to reuse these new appearance settings the next time the application is started.

For example, most widgets in the &xcenter; will accept fonts and colors dropped on them.

In detail, to change a font, do the following:

  1. Open the &link_lookfdr;.

  2. Double-click on the Font Palette in that folder to open it.

  3. Drag one of the fonts in the palette and drop it onto the window whose font you would like to change. For example, drag the font onto the "Pulse" widget in the &xcenter;.
To change a color, do the following:
  1. Again, open the &link_lookfdr;.

  2. Double-click on either the Solid Color Palette or the Mixed Color Palette in that folder to open it.

  3. Drag one of the colors in the palette and drop it onto the window to change the window's background color. If you would like to change the window's foreground color instead, hold down the Ctrl key while dragging.
