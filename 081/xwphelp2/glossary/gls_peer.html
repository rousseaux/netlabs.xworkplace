The File and Print Client is a subsystem of &os2; that allows you to share local resources with other computers on a local area network (LAN).

The File and Print Client is a stripped-down version of the networking components that ship with the server versions of &os2;. It consists of several services that can be started and stopped individually, depending on the functionality you need.

If these services have not yet been started, double-click on the "Start File and Print Client" object in the "Network Utilities" folder.

There are several ways you can configure the File and Print Client.

  • In the "&connectfdr;" folder on your Desktop, select "Sharing and Connecting".

  • From a command prompt, issue NET commands. Refer to the &os2; Command Reference: Introduction to OS/2 LAN Server Commands for details.
