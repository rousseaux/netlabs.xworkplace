If "Animate" is enabled, the &xcenter; frame will be animated when the &xcenter; is opened or closed.

This does several things:

  • It enables the system animation for the &xcenter; (which is only displayed if you also enabled "Animation" on the first "Window" page in the "Workplace Shell" notebook).

  • It also displays an extra animation to unfold the &xcenter; bar.

  • If "Auto-hide" is also enabled, the &xcenter; is more slowly hidden.
