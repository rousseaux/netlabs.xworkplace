The following widgets come with &xwp; by default:
  • Battery power
  • Date
  • Diskfree (condensed)
  • Diskfree (&warpcenter; style)
  • &xbutton;
  • Free physical memory
  • IP monitor
  • Object button
  • Pulse (CPU load)
  • Sentinel memory watcher
  • Window list
  • Time
  • Tray
  • &xbutton;
More widgets may be available from third parties because the &xcenter; supports plug-in DLLs.

To see a list of all supported widget classes, open the &xcenter; properties notebook and turn to the "Classes" page.
