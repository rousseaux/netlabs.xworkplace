If "Enable NumLock at startup" is checked, &xwp; will automatically turn on the "NumLock" key on the numeric keyboard while the Desktop is starting up. Changing this setting will also enable/disable NumLock immediately. The "NumLock" key can still be disabled manually, of course.

Note that this setting will only affect all programs running as windows in the Presentation Manager session (including windowed command prompts). It will not affect DOS windows, full-screen or &winos2; sessions. See "session types" for details.
