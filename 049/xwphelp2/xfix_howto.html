Wenn sie noch keine Sicherung von OS2.INI und OS2SYS.INI durchgeführt haben, beenden sie bitte xfix und führen eine Sicherung durch. Danach können sie xfix erneut starten.

  1. Der Anfang

    • Wählen sie Menüeintrag Aktionen->Suche ungültige Dateien. Dies führt zum Suchen nach Handles die auf gelöschte Dateien und Ordner verweisen. Die Suche könnte einen Moment dauern. Wenn sie beendet ist werden ungültige Handles wie folgt angezeigt: .
    • Wählen sie Menüeintrag Aktionen->Auto-Schutz. Dies führt zum Löschschutz von Handles die die WPS für eine korrekte Funktionsweise zwingend braucht. Diese Handles werden wie folgt dargestellt: oder .
    • Wählen sie Menüeintrag Sortieren nach->Bild. Daraufhin werden alle ungültigen Handles am Listenanfang angezeigt.
  2. Löschen ungültiger Handles

    xfix kann viele Probleme identifizieren aber nur einige von diesen beheben. Wenn ihr System stark beschädigt ist bestehen möglicherweise diese Probleme auch nach der Benutzung von xfix weiter.


    • Werfen sie für jedes ungültigen Handle einen Blick auf die "Referenzen" Spalte. Wenn sie eine Nummer sowie den Eintrag Abstrakte Objekte sehen haben sie ein Problem mit "Nicht-Dateisystem" Objekten wie zum Beispiel Schattenobjekten die verlorengegangen sind. Um diese wiederherzustellen, klicken sie mit gedrückter Maustaste 2 auf den entsprechenden Handleeintrag und wählen dann Abstrakte Objekte auf Arbeitsoberfläche verschieben. Wenn sie dies nicht tun wird das Objekt gelöscht sobald sie das Handle löschen.
    • Wählen sie alle als ungültig markierten Handles aus, dann wählen sie Löschen um sie zu löschen. Sie können hierzu die Entf Taste betätigen oder Löschen vom &popmenu; auswählen.
  3. Nicht benötigte Handles löschen

    Hier zeigt xfix seine wahre Stärke. Es ist gut möglich daß es massenhaft Handles gibt die die WPS nie benutzt und die nur das System verlangsamen. Auch wenn sie auf existierende Dateien zeigen sind die Handles oft unnötig und können gelöscht werden.


    • Wählen sie Sortieren nach->Langer Name um nach Laufwerk,Verzeichnis, Dateiname zu sortieren.
    • Finden sie Dateien und Ordner mit Schattenobjekten auf der Arbeitsoberfläche, dann wählen sie Schützen um ein Löschen der Handles zu verhindern. Dazu wählen sie entweder Schützen vom &popmenu; or sie betätigen Ctrl+S.
    • Wählen sie Sortieren->Dateiendung. Jetzt werfen sie einen Blick auf die "Kurzer Name" Spalte und scrollen herunter bis sie Dateien sehen die nach Dateiendung sortiert sind.
    • Jedes Handles das nicht schon geschützt ist kann zum Löschen ausgewählt werden. Löschen sie kein Handle wenn sie ein Schattenobjekt auf diese Datei/Ordner erzeugt haben.

      Vorrangig zum Löschen geeignet sind Handles auf .DLL und .ZIP Dateien. Programmierer können wahrscheinlich alle .C, .CPP sowie .H Einträge löschen. Wenn sie eine große Anzahl von Handles auf z.B. .JPG Dateien sehen und nicht sicher sind ob sie diese löschen können dann löschen sie zunächst nur ein oder zwei Einträge. Überprüfen sie nach dem Beenden von xfix ob dies irgendwelche negativen Auswirkungen hat -- wenn nicht starten sie xfix neu und löschen dann alle weiteren dieser Art.

    • Wenn sie dem Grundsatz folgen nie WPS Objekte für bestimmte Dateien oder Ordner anzulegen (zum Beispiel für Ordner "DOWNLOADS" oder "TEMP") dann können sie die Handles für diese Dateien in diesen Ordnern löschen. Wählen sie noch einmal Sortieren->Langer Name um nach Dateipfad zu sortieren, finden sie den entsprechenden Ordner auf und löschen sie alle Handles mit dem entsprechenden Dateipfad.
  4. Speichern und Beenden

    • Die Statusleiste am unteren Fensterrand sollte "0 ungültig" and "0 verwaist" anzeigen. Wenn nicht, wählen sie Sortieren->Bild. Die ungültigen Handles werden am Listenanfang angezeigt -- löschen sie diese. Die verwaisten Handles sind als nächstes an der Reihe -- schützen sie diese vor dem Löschen.
    • Select xfix->Zurück in OS2SYS.INI schreiben...
    • Sie werden gefragt ob sie die Handles entfernen wollen die sie zum Löschen ausgewählt haben. Wenn eine Warnung angezeigt wird daß verwaiste Handles existieren, wählen sie Abbrechen und beheben sie das Problem. Wenn nicht, wählen sie Ja.
    • Danach werden sie gefragt ob sie wirklich die Änderungen abspeichern wollen. Lesen sie eventuelle Warnungen, wenn sie sicher sind daß sie keine neuen Probleme erzeugt haben antworten sie mit Ja.
    • Schließen sie das Hauptfenster um xfix zu beenden.
