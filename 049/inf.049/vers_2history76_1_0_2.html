Pure bugfix release off the 1.0 branch. "Bug #" numbers refer to the official XWorkplace bugtracker at http://xtracker.netlabs.org.

XWorkplace 1.0.2 (January 28, 2004)

Bugfixes:

  • "Restart" Desktop stalled at "Closing WarpCenter". Fixed by Paul Ratcliffe. (Bug #302)

  • Drag&Drop of URLs from Mozilla hung WPS bad because of infinite recursion in file-system refresh. Fixed. (Bug #531)

  • Bart van Leeuwen fixed status bar conflict when WPS-Wizard 0.4 was installed. (Bug #481)

  • Bart van Leeuwen enlarged a text buffer in the "Pulse" widget to fix a problem with Dutch NLS. (Bug #490)

  • Icon page was missing in in File and Print Client Resource Browser object. Fixed by Paul Ratcliffe. (Bug #514)

  • Fixed Korean codepage in encoding engine from 951 to 949.

  • Tooltips followed the mouse if moved quickly from XCenter. Fixed by Paul Ratcliffe. (Bug #455)

  • Fixed clipped text and button widths in panic dialog. (Bug #361)

  • Kernel object, Errors page, Drive for POPUPLOG.OS2 still showed incorrect drive (always the first HD on your system). Fixed by Paul Ratcliffe. (Bug #37)

  • Selecting original -> delete from a shadow context menu sometimes deletes the wrong object if replacement delete is enabled. Fixed by Paul Ratcliffe. (Bugs #8, #324)

  • "Diskfree (condensed)" widget had weird display if a drive could not be accessed. Fixed by Paul Ratcliffe. (Bug #525)

  • "Diskfree (condensed)" widget issued two open commands when double-clicked. Fixed. (Bug #398)

  • Fixed codepage problems with file types import/export. Note that the XML for exporting non-ASCII characters has changed, so you might run into problems importing files exported by earlier XWorkplace versions. (Bugs #13, #376)

  • Fixed some documentation errors. (Bugs #372, #370)

  • XCenter didn't remember last tray between restarts. Fixed by Bart van Leeuwen. (Bug #10)
