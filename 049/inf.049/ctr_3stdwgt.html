Standardmäß besitzt das &xcenter; folgende Widgettypen:
  • Das "X-Button" Widget. Es zeigt das &xwp;-"X"-Logo in der &xcenter;-Leiste an. Durch Klicken auf diese Schaltfläche öffnet sich ein Menü, welches den Zugriff auf die Funktionen "Wartemodus aktivieren", "WPS neu starten" und &xshutdown erlaubt. Außerdem wird der Ordnerinhalt der Arbeitsoberfläche angezeigt.

  • Objektschaltflächen. Diese repräsentieren beliebige Objekte der Arbeitsoberfläche. Dieser Widgettyp wird im Menü "Widget hinzufügen" nicht angezeigt. Um ein solches Widget hinzuzufügen, ziehen Sie einfach ein Objekt auf die &xcenter;-Leiste, wie es auf der vorigen Seite gezeigt wird.

    Handelt es sich bei dem Objekt um einen Ordner, wird ähnlich den XFolder- Orderinhaltsmenüs dessen Inhalt angezeigt, wenn Sie darauf klicken:

  • Das Widget "CPU-Last", das auch im oberen Bildschirmphoto sichtbar ist, zeigt die aktuelle CPU-Last als Prozentwert an. Zudem legt es eine graphische Darstellung der vergangenen CPU-Aktivität an.

  • Dann gibt es noch verschiedene Überwachungswidgets, welche Informationen über das System anzeigen. Momentan sind die Widgets Uhr und "Freier Speicher":

  • Schließlich zeigt das Widget "Fensterliste" den Inhalt der Fensterliste des Presentation Manager an. Im Gegensatz zu dieser Liste wird das momentan aktive Fenster als gedrückte Schaltfläche angezeigt, und Sie können auf die Einträge klicken, um die entsprechenden Fenster zu aktivieren. Nebenbei unterstützt diese Fensterliste auch das Filtern von Einträgen (klicken Sie mit der rechten Maustaste auf die Fensterliste und wählen Sie "Einstellungen").
