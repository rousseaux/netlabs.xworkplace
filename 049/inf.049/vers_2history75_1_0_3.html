Bugfix release off the 1.0 branch and a few simple new features. "Bug #" numbers refer to the official XWorkplace bugtracker at http://xtracker.netlabs.org.

&xwp; 1.0.3 (February 23, 2005)

Bugfixes:

  • XPager won't change Desktops when certain apps are running (Homepage Publisher). Fixed by Bart van Leeuwen. (Bug #524)

  • "Restart Desktop" stalled at "Closing WarpCenter" due to subclassed WarpCenter. Fixed by Paul Ratcliffe. (Bug #302)

  • Title clash dialog now only appears when creating filesystem objects. Abstract objects (typically shadows) can exist with the same name without clashing. Fixed by Paul Ratcliffe. (Bug #563)

  • ObjectIDs can now be deleted in Properties, Icon tab, Details. Fixed by Paul Ratcliffe. (Bug #580)

  • Module notebook page reports wrong application type for some executable headers. Fixed by Paul Ratcliffe. (Bug #583)

New Features:

  • Added decoding of OPEN and SHADOWID setup string items in Properties, Icon tab, Details. Added by Paul Ratcliffe.

  • Allow / in replace string in XMakeini. Added by Alfredo Fernandez Diaz.

  • XPager: Add Windows keys as an option to switch desktops. Added by Knut St. Osmundsen. (Feature request #588)
