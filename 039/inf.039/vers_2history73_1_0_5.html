Bugfix release off the 1.0 branch. "Bug #" numbers refer to the official XWorkplace bugtracker at http://xtracker.netlabs.org.

&xwp; 1.0.5 (July 29, 2006)

Bugfixes:

  • Tidied up the Recovery Choices options on the Desktop Properties Archives tab. Fixed by Paul Ratcliffe.

  • Double clicking the XCenter Date/Time widgets brought up the System Clock first rather than the eCS Clock in eCS. Fixed by Paul Ratcliffe.

  • The tooltip for the XCenter Date widget now has the time rather than the date as the tooltip text. Fixed by Paul Ratcliffe.

  • Split view folder status bars now update correctly when selecting/deselecting objects. Bug 326. Fixed by Paul Ratcliffe.

  • Cured cosmetic problems when resizing the panes of a split view folder. Bug 228. Fixed by Paul Ratcliffe.

  • Corrected missing objects during installation. Bug 695. Fixed by Paul Ratcliffe.

  • Fixed Association problem when Turbo Folders/Extended Associations were enabled. Bug 14. Fixed by Paul Ratcliffe.

  • Fixed missing WPS file filters during first Import with Turbo Folders/Extended Associations. Bug 399. Fixed by Paul Ratcliffe.

  • Fixed XPager virtual desktop switching on top/bottom borders. Bug 406. Fixed by Paul Ratcliffe.

  • Fixed Exporting of Extended Associations. Bug 702. Fixed by Paul Ratcliffe.

  • Fixed WarpCenter stuck in XCenter's Window List widget. Bug 11. Fixed by Paul Ratcliffe.

  • Fixed partially broken Undo in XWP Setup notebook, Features tab. Fixed by Paul Ratcliffe.

  • NLS Settings are now correctly fetched from the Country object on W4 FP13+ systems. Bug 614 again. Fixed by Paul Ratcliffe.

  • Fixed Default and Undo on Folder Views tab of Workplace Shell object. Fixed by Paul Ratcliffe.

  • Fixed traps when XFldDisk class was not registered. Fixed by Paul Ratcliffe.

  • Fixed broken Bitmap display in Light Table folders. Bug 371. Fixed by Paul Ratcliffe.

New Features:

  • NLS packages are now detected during install. Bug 686. Implemented by Christian Hennecke.

  • XWP has some updated icons courtesy of David Graser.

  • ACPI power-off option is now available with XShutdown. Implemented by Paul Ratcliffe.
