Siete infastiditi dal fatto che cliccando su di un oggetto URL od un file HTML viene sempre avviata una nuova istanza di Mozilla o di Netscape, anche se c'Š una copia gi… in esecuzione? Si tratta senza dubbio di uno spreco di tempo, memoria e risorse del sistema.

L'Interfaccia DDE a Netscape Š la soluzione. Ô possibile associarla a file HTML e oggetti URL, in modo che facendo doppio clic su di essi venga avviato il browser prescelto, oppure, se gi… aperto, il file HTML o l'oggetto URL vengano subito passati all'istanza in esecuzione.

NetscapeDDE funziona correttamente con Netscape 2.02, 4.04, e 4.61 per &os2;.

Funziona inoltre con tutte le versioni di Mozilla dotate di supporto DDE; almeno, quindi, dalla versione 1.0.0 in su. Si veda la sezione Trucchi & suggerimenti per istruzioni su come sfruttare il supporto DDE.
