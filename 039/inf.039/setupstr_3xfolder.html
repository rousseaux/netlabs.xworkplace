Classe: XFolder
Le seguenti stringhe di impostazioni possono essere usate nello stesso modo delle classiche stringhe descritte nella pagina precedente; comunque esse hanno effetto solo se XFolder Š correttamente installato.
ACCELERATORS={YES|NO|DEFAULT}
Stabilisce, per una singola cartella, se questa debba o meno rispondere alle scorciatoie di cartella. Quest'impostazione corrisponde a quella sulla pagina "XFolder" del blocco impostazioni. YES o NO imposter… questo attributo, senza riguardo a ci• che era specificato nelle impostazioni globali, mentre DEFAULT imposter… il valore globale. Consultare "Tasti rapidi cartella".

ALWAYSSORT={YES|NO|DEFAULT}
Stabilisce, per una singola cartella, se questa manterr… o meno il suo metodo di ordinamento. A differenza delle standard WPS folder setup string, la nuova interpretazione di XFolder accetta anche "DEFAULT" come parametro, che riporter… l'impostazione a quanto specificato sulla pagina "Ordinare" dell'oggetto "Sistema".

DEFAULTSORT=crit
Imposta il criterio di ordinamento predefinito per una singola cartella a un valore diverso dal criterio usato globalmente. crit pu• essere uno dei seguenti:
-1 per tipo
-2 per nome
-3 per classe oggetto
-4 per estensione
255 ripristina il criterio globale

Notare che in dipendenza dalla classe di ordinamento della cartella, potrebbero essere supportati anche valori tra 0 e 100. Quei valori corrispondono agli indici delle colonne dei dettagli degli oggetti nella Vista dettagli e sono specifici della classe. Si provino i valori riportati in un blocco impostazioni di una cartella.

FAVORITEFOLDER={YES|NO}
Stabilisce per una singola cartella se i suoi contenuti possano essere visti da tutti i menu contestuali sul sistema. Questa impostazione corrisponde a quella sulla pagina di impostazioni "XFolder". Il valore predefinito Š NO. Non ci sono valori globali per quest'impostazione. Si consulti "Folder content functions" per dettagli.

FULLPATH={YES|NO|DEFAULT}
Stabilisce, per una singola cartella, se il suo percorso completo debba essere riportato nella barra del titolo della cartella. Quest'impostazione corrisponde a quella sulla pagina di impostazioni "XFolder". YES o NO impostano quest'attributo senza riguardo per quanto specificato nelle impostazioni globali, mentre DEFAULT porter… l'impostazione al valore predefinito. Consultare "Full path in window titles" per dettagli.

QUICKOPEN={YES|NO|IMMEDIATE}
Stabilisce, per una singola cartella, se questa dovr… essere prepopolata all'avvio della Scrivania. Quest'impostazione corrisponde a quella sulla pagina di impostazioni "XFolder". YES o NO impostano l'attributo per la cartella. Il valore assunto Š NO. Non ci sono valori globali per quest'impostazione. Consultare "Cartelle ad apertura rapida" per i dettagli.

Come comando speciale QUICKOPEN=IMMEDIATE popola immediatamente la cartella. Pu• essere usato in uno script REXX per popolare una cartella senza doverla aprire. QUICKOPEN=IMMEDIATE, quindi, non Š tanto un'impostazione quanto un comando da eseguire sulla cartella.

SNAPTOGRID={YES|NO|DEFAULT|EXEC}
Sets for an individual folder whether the "Snap to grid" item will appear in the folder's context menu. This setting corresponds to the one on the "XFolder" notebook settings page. YES or NO will set this setting for this folder no matter what was specified in the global settings, while DEFAULT will set this setting to the global value. EXEC will execute the "Snap to grid" function on the folder, just as if you had selected it from the context menu. EXEC is thus no setting, but a command to be performed on a folder. See "Snap to grid" for details.

SORTFOLDERSFIRST={YES|NO|DEFAULT}
Sets for an individual folder whether folders will always be sorted in first. This setting corresponds to the one on the folder's "Sort" notebook settings page. YES or NO will set this setting for this folder no matter what was specified in the global settings, while DEFAULT will set this setting to the global value.

SORTNOW=crit
This will sort all currently open views of an individual folder without changing a folder's setting (as opposed to DEFAULTSORT). This is not really a "setting", but rather a command to be performed on a folder. crit has the same meaning as with DEFAULTSORT.

STATUSBAR={YES|NO|DEFAULT}
Sets the visibility state for an individual folder status bar. This setting corresponds to the one on the "XFolder" notebook settings page. YES or NO will set this setting for this folder no matter what was specified in the global settings, while DEFAULT will set this setting to the global value. See "Folder status bars" for details.
