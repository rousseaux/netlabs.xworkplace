Classe: XFolder
XFolder permette di selezionare una parte degli oggetti contenuti nella cartella secondo una specifica tipica.

Quest'operazione viene compiuta tramite la voce di menu "Selezionare per nome", presente, in Warp 3, nel menu "Selezionare"; su Warp 4, questa voce di menu si trova nel menu "Visualizzare".

Scegliendo "Selezionare per nome" si apre una finestra di dialogo dalla quale Š possibile inserire una specifica di file come "*.txt" e selezionare o deselezionare file in base a questa specifica.

Come valore assunto, XFolder definisce anche Ctrl+S come tasto rapido cartella per questa funzione.
