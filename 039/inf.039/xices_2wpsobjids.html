Le ID Oggetto seguenti sono predefinite all'installazione di &os2; e dovrebbero perci• essere accessibili in ogni sistema (a meno che l'utente non abbia cancellato l'oggetto corrispondente).
  • Cartelle di sistema:

    <WP_NOWHERE>: cartella nascosta (dove si trova il Pannello di avvio, "?:\Nowhere")
    <WP_DESKTOP>: la Scrivania correntemente attiva
    <WP_OS2SYS>: cartella Sistema di &os2;
    <WP_TEMPS>: cartella Modelli
    <WP_CONFIG>: cartella Impostazione del sistema
    <WP_START>: cartella Avvio
    <WP_INFO>: cartella Informazioni
    <WP_DRIVES>: cartella Unit…
    <WP_PROMPTS>: cartella Richieste comandi
    <WP_TOOLS>: cartella Utilities
    <WP_GAMES>: cartella Giochi

  • Altri oggetti di sistema:

    <WP_VIEWINF>: programma VIEW.EXE
    <WP_SYSED>: editor di sistema &os2;
    <WP_ICON>: editor di icone
    <WP_PULSE>: Pulse
    <WP_CLIPV>: visualizzatore Appunti

    <WP_OS2FS>: sessione schermo pieno &os2;
    <WP_OS2WIN>: sessione in finestra &os2;
    <WP_DOSFS>: sessione schermo pieno DOS
    <WP_DOSWIN>: sessione in finestra DOS
    <WP_WINFS>: sessione Win-&os2; a schermo pieno
    <WP_WIN2WIN>: sessione Win-&os2; in finestra

    <WP_KEYB>: impostazioni tastiera
    <WP_MOUSE>: impostazioni mouse
    <WP_SYSTEM>: impostazioni sistema
    <WP_CNTRY>: impostazioni nazione
    <WP_SOUND>: impostazioni suono
    <WP_SPOOL>: spooler

    <WP_FONTPAL>: elenco caratteri
    <WP_HIRESCLRPAL>: elenco colori 1
    <WP_LORESCLRPAL>: elenco colori 2
    <WP_SCHPAL>: elenco schemi

    <WP_VIEWER>: visualizzatore finestre ridotte
    <WP_SHRED>: trinciatrice
    <WP_CLOCK>: orologio di sistema

Oltre a queste, &xwp; introduce le ID Oggetto seguenti (si noti che sono tutte cambiate dalla V0.9.2):

<XWP_CONFIG>: la cartella di configurazione (principale) di XFolder
<XWP_STARTUP2>: la cartella di avvio &xwp;
<XWP_SHUTDOWN2>: la cartella chiusura &xwp;
<XWP_KERNEL>: l'oggetto "Kernel &os2;"
<XWP_WPS>: l'oggetto "Workplace Shell"
<XWP_REF>: l'oggetto "Guida ad XWorkplace"
