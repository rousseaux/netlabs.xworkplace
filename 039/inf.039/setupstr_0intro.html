Per cambiare le impostazioni di un certo oggetto usando il REXX, si usino le seguenti funzioni RexxUtil: 
rc = SysSetObjectData(name, setup)

dove:

name Š l'oggetto che si desidera cambiare (riferito da un percorso completo o un ID di oggetto, come <WP_DESKTOP>);

setup Š una stringa di impostazione di WPS, che ha sempre sintassi keyword=data. Parecchie impostazioni possono essere combinate separandole da un punto e virgola (;).

Esempio: 

  if SysSetObjectData(directory(),"NOMOVE=YES;DEFAULTVIEW=ICON") then
    Say "Impostazioni cambiate!"

Questo cambier… le impostazioni di cartella per la directory corrente, cosa utilissima per script chiamati da XFolder nel caso si sia gi… passati automaticamente ad una directory differente.

Consultare le Informazioni sul REXX di &os2; per dettagli.
