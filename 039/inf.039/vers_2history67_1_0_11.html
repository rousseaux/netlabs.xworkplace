&xwp; 1.0.11 (October 27, 2016)

All changes listed below are the product of Rich Walsh.

Changes:

  • Xview: splitview has been enhanced and renamed Xview. Among the new features available are:
    • a new details view mode to complement the existing icon view
    • menu-accessible view options that are specific to Xview
    • the ability to set Xview as the system default folder view
    In addition, several pre-existing bugs in splitview have been fixed.

  • NLS Packaging: for each supported language (DE/EN/IT/JA/NL) the XWP build system now creates a complete package with all needed components. It is no longer necessary for users to install a base package and then another package that supports their language.

  • Build System Customization: the XWP build system has been extensively revised to permit builders to create their own customized "full" and "lite" builds of XWP without making any changes to the source code. Full details of these changes along with how-to instructions can be found in "buildme.xwp" and "ecs\buildme.ecs" located within the XWP source tree.
