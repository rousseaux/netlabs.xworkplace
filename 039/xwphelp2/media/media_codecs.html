"Codec" Š un acronimo per "compressore/decompressore". Un codec Š necessario per comprimere o decomprimere i contenuti di un file audio o video. Mentre il formato di un file (ad es. .AVI o .WAV) viene gestito da un IOProc (vedere pagina precedente), i contenuti di un file sono gestiti da un codec.

Per esempio, MMPM/2 viene fornito con un IOProc per il formato file .AVI, ma non ha abbastanza codec per comprendere tutti i formati di compressione solitamente utilizzati per i file AVI.

Il "FourCC" (codice di quattro caratteri) di un codec deve combaciare quello del IOProc. Lo stesso codec pu• essere installato diverse volte per diversi IOProcs. Per esempio, il codec "Intel Indeo" Š installato nell'IOProc per i file AVI. Ma se si disponesse di un nuovo IOProc per qualche altro formato video (ad es. il ".MOV" di QuickTime), si potrebbe utilizzare lo stesso codec per i file .MOV, dato che il formato di compressione Š lo stesso! Soltanto il formato del file cambia.
