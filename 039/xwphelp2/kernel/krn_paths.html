Questa pagina permette di controllare i percorsi di sistema pi— importanti in &cfgsys;, verificandone eventuali parti non valide.

Nota: si abbia estrema cautela nel modificare questi percorsi. Una sola modifica non valida potrebbe rendere inutilizzabile tutto il sistema.

Premere "Applica" per salvare nel file &cfgsys; le modifiche effettuate. Vengono salvati i cambiamenti effettuati in tutti i percorsi presenti nella finestrella a caduta, non solo quelli effettuati sul percorso selezionato. E' necessario riavviare il computer perch‚ le modifiche effettuate abbiano effetto. Il solo riavvio della Scrivania non Š sufficiente.

La finestra a caduta "Percorso di sistema da verificare" presenta l'elenco dei percorsi di sistema che possono essere modificati tramite questa pagina. Si veda "Percorsi di sistema" per spiegazioni ulteriori.

Selezionando un percorso, le directory ad esso associate nel file &cfgsys; vengono visualizzate nella finestra sottostante. E' possibile selezionare gli elementi nella finestra ed usare i bottoni per modificarli. Questa operazione dovrebbe essere sufficientemente autoesplicativa.

L'opzione "Selez. non validi" seleziona tutte le directory non valide del sistema. Si possono poi rimuovere premendo il tasto "Cancella".
