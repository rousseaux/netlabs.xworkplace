La pagina "Programmi controllo" consente di visualizzare i driver che sono attualmente caricati dal file &cfgsys; e fornisce una comoda interfaccia per visualizzare e modificare i relativi parametri.

I programmi di controllo sono ordinati in categorie. Espandenre un elemento dell'albero per vederli tutti. Sceltone uno dall'albero a sinistra, sulla destra vengono mostrate le informazioni relative.

  • Le informazioni sulla "Versione" e sul "Fornitore" sono lette direttamente all'interno del file, analogamente a quanto fa il programma BLDLEVEL.

  • Il campo "Parametri" riporta i parametri impostati per questo driver nel file &cfgsys;.
Cliccando con il tasto destro del mouse nella finestra di sinistra, sul nome di un driver se ne apre il relativo &popmenu;e, dal quale si pu• ottenere un aiuto supplementare:
  • Se il driver fa parte della installazione standard di &os2;, selezionando "Mostra aiuto in Riferimento per i comandi OS/2" si visualizza la pagina relativa al driver nel Riferimento per i comandi &os2; (CMDREF.INF).

  • Il database ConfigTool, pacchetto gratuito disponibile separatamente, fornisce informazioni pi— approfondite per numerosi driver. Selezionando "Mostra aiuto dal database ConfigTool" viene visualizzato l'appropriato pannello di aiuto. Questa voce di menu Š abilitata solo se il file .INF di ConfigTool, CFGDAT.INF, si trova lungo il percorso di sistema specificato dalla variabile ambiente BOOKSHELF nel file &cfgsys;.

  • Alcuni driver dispongono di un proprio file di aiuto e spesso si possono trovare informazioni utili anche in file di aiuto non immediatamente riconducibili ad uno specifico driver. Selezionando "Cerca in tutti i file di aiuto" si avvia una ricerca per quel dato nome di driver in tutti i file di aiuto presenti nei percorsi BOOKSHELF e HELP. Al termine della ricerca, sar… possibile scegliere tra i file trovati quello che pi— interessa.
