Questa finestra di dialogo permette di configurare JFS.IFS.

Per una panoramica sui file system, selezionare una delle voci seguenti:

  • File system
  • Cache dei file system
Premendo "OK" le nuove selezioni sono copiate nel campo "Parametri" della pagina "Programmi controllo" nell'oggetto impostazioni "Kernel &os2;". Questo tuttavia non modifica ancora il file &cfgsys;. Per modificarlo, dopo aver premuto "OK" in questa finestra, premere anche "Applica" dalla pagina "Programmi controllo"

Per una spiegazione dettagliata di ciascun campo selezionare:

  • Dimensione cache
  • Autocheck
  • Lazy write
