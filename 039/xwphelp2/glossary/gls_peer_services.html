Generalmente i servizi &link_peer; attivi nel sistema sono i seguenti:
  • Il servizio REQUESTER, che permette al computer di vedere le altre risorse nella LAN. "Requester" Š in effetti da intendersi come sinonimo per "client". Una volta avviato il Requester, file e stampanti remote presenti nella LAN diventano accessibili. Inoltre si possono assegnare lettere di unit… del proprio sistema a cartelle remote messe in condivisione da altri computer.

  • Il servizio PEER Š una versione ridotta del servizio server che fa parte delle versioni server di &os2;. Una volta avviato il servizio Peer, il proprio computer potr… condividere cartelle locali, file e stampanti con gli altri computer della LAN.

    Questo servizio per funzionare richiede che sia attivo anche il servizio REQUESTER.

Utilizzare la linea comando per gestire i servizi in dettaglio:
  • Per avviare un certo servizio digitare NET START servizio; per esempio: NET START REQUESTER.

  • Per interromente un certo servizio digitare NET STOP servizio; per esempio: NET STOP REQUESTER.

  • Per vedere un elenco dei servizi attivi al momento digitare NET START, senza specificare nomi di servizi.
