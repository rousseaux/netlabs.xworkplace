Gli operatori comando sono caratteri speciali interpretati dai processori di comando come comandi specifici piuttosto che normali caratteri. Questi operatori possono essere scritti in una linea di comando, o inclusi in un file di comandi o batch.

Se si desidera usare come carattere normale un carattere normalmente usato come operatore comando, ä necessario dargli un escape facendolo precedere dal carattere ^. Alcuni operatori comando sono validi solo nelle finestre comandi OS/2, non nelle sessioni DOS.

Segue un elenco degli operatori comando nell'ordine in cui vengono riconosciuti dagli interpreti di comandi:

  • Il carattere ^ "fuga" (escape) un operatore comando, per farlo riconoscere come un normale carattere in ingresso o in uscita (solo sessioni OS/2). Per esempio, per mostrare sullo schermo la seguente frase: 
    INSERIRE ETA` & NUMERO PREVIDENZA SOCIALE ==>
    si scriva: 
    INSERIRE ETA` ^& NUMERO PREVIDENZA SOCIALE ==^>

  • > ridirige l'uscita verso un file o un dispositivo. Se il file esiste viene sovrascritto.

    >> ridirige l'uscita verso un file e, se questo esiste, appende l'uscita alla fine del file preesistente.

    < recupera e ridirige l'input dal dispositivo standard di ingresso.

    La sorgente d'ingresso e l'uscita di un programma sono predefinite e identificate da un numero (chiamato handle). Esistono i seguenti tre handle:

    • Standard input, chiamato STDIN, con handle 0. Per default corrisponde alla tastiera.
    • Standard output, chiamato STDOUT, con handle 1. Per default corrisponde allo schermo.
    • Viene usata un'uscita differente per gli errori (STDERR), con handle 2. Normalmente ä associata allo schermo, ma ciï permette di ridirigere i messaggi d'errore separatamente.
    L'ingresso e l'uscita di un programma possono essere ridiretti solo se il programma sta leggendo da STDIN e sta scrivendo su STDOUT, incluso STDERR. Se il programma non implementa queste convenzioni, per esempio se non ä un programma in modo testo o se legge e scrive direttamente sull'hardware, la redirezione non funziona.
