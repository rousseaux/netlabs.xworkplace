Nell'ambito di una rete locale (LAN) un server Š un computer che condivide le sue risorse con altri computer, i client.

In una LAN il server Š generalmente una macchina potente con un sistema operativo dalle funzionalit… estese, specificatamente disegnato per gestire un gran numero di richieste di rete e utenti multipli, ciascuno con diritti di accesso distinti. I computer client di solito non condividono risorse, limitandosi ad accedere a quelle sul server.

Al contrario, in una rete "peer" ogni computer si comporta sia come server che come client.
