Selezionare questa voce di menu per far sç che &xwp; copi negli Appunti il nome file dell'oggetto.

Se si preme contemporaneamente anche il tasto Maiusc, &xwp; copia l'intero percorso.

Se nella cartella si sono selezionati pió oggetti tutti i nomi dei file selezionati saranno copiati, separati da spazi.

Come valore assunto &xwp; assegna il tasto rapido Ctrl+Ins per copiare i nomi dei file senza il percorso completo.

E' possibile rimuovere questa voce di menu dalle pagine "Menu" nell'oggetto "Workplace Shell".
