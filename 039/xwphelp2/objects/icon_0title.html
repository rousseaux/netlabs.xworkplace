Il campo "Nome" mostra il nome dell'oggetto. Questo Š il nome che, assieme all'icona, Š visualizzato nelle cartelle. Editare questo campo per modificare il titolo dell'oggetto.

Il nuovo titolo viene visualizzato al di sotto dell'icona, su una o pi— righe. Premere il tasto Invio per indicare l'inizio di una nuova riga. E' possibile usare qualsiasi carattere, spazi inclusi, eccetto il backslash (\).

Se l'oggetto rappresenta un file fisicamente esistente o una cartella, modificando il titolo dell'oggetto si modifica anche il nome del file relativo ("nome reale"). A seconda del file system su cui l'oggetto si trova sono da tenere presenti possibili limitazioni. Si veda "Nomi reali" per i dettagli.
