I file interscambio immagini (PIF) contengono le istruzioni necessarie a definire un'immagine. Il PIF Š un vecchio formato, una volta utilizzato per scambiare file di immagini tra i programmi di grafica.

Come valore assunto i file PIF sono associati con il Visualizzatore immagini di &os2;, che permette di visualizzarli e stamparli.
