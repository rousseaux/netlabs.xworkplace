Un "oggetto tipo di carattere" rappresenta un file font installato nel sistema. Gli oggetti tipi di carattere esistono solo all'interno della cartella Tipi di carattere aperta.

Nota: In questo contesto con il termine "font" ci si riferisce ad una variante font, rappresentata fisicamente da un file font. Nella maggior parte dei casi ogni famiglia di tipi di carattere Š formata da diverse varianti. Per esempio, la famiglia "Times New Roman" Š composta da quattro file font (per "Regular", "Bold", "Italics" e "Bold Italics" rispettivamente). Solo se per un certo formato quella variante manca (se, per es., nel word processor si usa del testo in grassetto, ma la variante grassetto del tipo di carattere corrente manca), &os2; imiter… i formati grassetto e corsivo col suo motore grafico interno (generalmente con risultati modesti).

Anche se la maggior parte delle applicazioni (nonch‚ la finestra tipi di carattere standard &os2;) mostrano solo i nomi delle varie famiglie e trovano automaticamente il file tipo di carattere corretto per il formato selezionato, la cartella Tipi di carattere di &xwp; elenca separatamente tutte le varianti font installate. E' comunque possibile vedere nella apposita colonna le rispettive famiglie di appartenenza.

E' possibile installare nuovi tipi di carattere trascinando i file font relativi nella cartella Tipi di carattere. E' poi possibile disinstallarli selezionando "Disinstalla..." dal &popmenu;e dei vari oggetti tipo di carattere.
