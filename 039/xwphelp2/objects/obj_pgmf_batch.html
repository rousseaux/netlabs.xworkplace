I file batch ed i file comandi, detti anche "script", sono file di testo semplice contenenti una serie di comandi per un processore comandi, che vengono processati in sequenza quando il file viene eseguito.

File batch e file comandi sono eseguiti da uno dei processori comandi; quale verr… usato dipende dall'estensione del file:

  • I file con estensione .BAT (batch file) sono eseguiti in sessioni DOS attraverso il processore comandi DOS, COMMAND.COM. Si noti che il processore comandi &os2; (CMD.EXE) riconosce anche l'estensione .BAT. Se da una richiesta comandi si avvia un file .BAT, questo verr… automaticamente lanciato in una sessione DOS.
I file batch ed i file comandi sono utili per automatizzare certe funzioni. Se, per esempio, ci si trova spesso ad inserire la medesima serie di istruzioni dalla richiesta comandi, si pu• decidere di creare un file batch o comandi con questa sequenza ed usare poi questo script invece di digitare manualmente le varie istruzioni. File di questo tipo si possono comodamente realizzare con l'&eexe; di &os2;.
