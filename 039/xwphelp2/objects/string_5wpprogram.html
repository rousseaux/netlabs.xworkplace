WPProgram Š la classe WPS per gli &link_progobjs;. (si veda Classi WPS per i dettagli). Oltre alle stringhe impostazioni proprie della classe WPObject, la classe WPProgram supporta le stringhe seguenti:

ASSOCFILTER=filtri

Imposta i filtri per i file dati da associare all'oggetto programma. Filtri multipli vanno separati con virgole.

Nota: se si abilitano le associazioni estese di &xwp; il filtro viene registrato nei file INI, ma la modifica non ha effetto finch‚ le associazioni estese non vengono disabilitate. &xwp; infatti ignora completamente le associazioni WPS attraverso i filtri.

ASSOCTYPE=tipi

Imposta i tipi per i file dati associati all'oggetto programma. Tipi multipli vanno separati con virgole.

EXENAME=nomefile

Imposta l'eseguibile cui punta l'oggetto programma. Deve essere un eseguibile ben qualificato (cioŠ per cui si Š specificato l'intero percorso) o presente nel percorso PATH di sistema, altrimenti l'avvio del programma fallir….

MAXIMIZED={YES|NO}

Determina se l'eseguibile cui punta l'oggetto programma deve essere avviato alla dimensione massima o no. Questa impostazione generalmente non ha effetto sui programmi PM.

MINIMIZED={YES|NO}

Determina se l'eseguibile cui punta l'oggetto programma deve essere avviato minimizzato o no. Questa impostazione generalmente non ha effetto sui programmi PM.

PROGTYPE=tipo

Imposta esplicitamente il tipo di sessione per l'eseguibile cui punta l'oggetto programma. tipo deve essere uno dei seguenti:
  • DOSMODE: modalit… DOS stand-alone.
  • FULLSCREEN: modalit… &os2; a schermo pieno.
  • PM: Presentation Manager (PM).
  • PROG_30_STD: modalit… &winos2;, compatibilit… standard e schermo pieno.
  • PROG_31_ENH: modalit… &winos2;, compatibilit… avanzata e schermo pieno.
  • PROG_31_ENHSEAMLESSCOMMON: finestra &winos2; in sessione comune &winos2;, compatibilit… avanzata.
  • PROG_31_ENHSEAMLESSVDM: finestra &winos2; in sessione separata, modalit… compatibilit… avanzata.
  • PROG_31_STD: &winos2;, modalit… schermo pieno, compatibilit… standard.
  • PROG_31_STDSEAMLESSCOMMON: finestra &winos2; in sessione comune &winos2;, compatibilit… standard.
  • PROG_31_STDSEAMLESSVDM: finestra &winos2; in sessione separata, modalit… compatibilit… standard.
  • SEPARATEWIN: finestra &winos2; in VDM separata.
  • VDM: DOS schermo pieno.
  • WIN: &winos2; schermo pieno.
  • WINDOWABLEVIO: &os2; in finestra.
  • WINDOWEDVDM: DOS in finestra.
  • WINDOWEDWIN: finestra &winos2;.

NOAUTOCLOSE={YES|NO}

Determina se la sessione rester… aperta al termine del programma. Questa impostazione non influenza i programmi PM. Il valore assunto Š NO.

PARAMETERS=parametri

Imposta i parametri per l'eseguibile cui punta l'oggetto programma. Pu• contenere le macro sostitutive standard per gli oggetti programma.

SET variabile=valore

Imposta la variabile ambiente per la sessione dell'eseguibile.

Si noti che &os2; usa le variabili ambiente anche per impostare certi valori DOS nelle sessioni DOS e &winos2;.

STARTUPDIR=percorso

Imposta la cartella di lavoro in cui avviare la sessione.
