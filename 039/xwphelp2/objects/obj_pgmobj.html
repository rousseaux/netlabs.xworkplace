Cliccare due volte sull'oggetto programma per avviare il programma ad esso associato.

L'oggetto programma in realt… Š solo un rinvio al file programma: aprendo il blocco appunti impostazioni dell'oggetto programma Š infatti chiaramente visibile il percorso del file programma cui l'oggetto punta.

L'oggetto programma permette anche di controllare una serie di impostazioni aggiuntive relative all'avvio del programma, come l'indirizzario di lavoro, i parametri ed il tipo di sessione.

In &os2; gli oggetti programma servono poi per associare i &link_datafiles; alle applicazioni. In questa maniera cliccando due volte su un file dati lo si apre automaticamente con il programma ad esso associato.

Per creare un nuovo oggetto programma per un file programma:

  • aprire la &link_tempsfdr;, partendo dal relativo modello trascinare nella posizione desiderata un nuovo oggetto programma e inserire i dati necessari nel suo blocco appunti impostazioni;

  • oppure, pi— comodamente, trascinare il file programma in una visualizzazione cartella (o sulla Scrivania): verr… creato automaticamente un nuovo oggetto programma con tutti i dati necessari gi… impostati.
