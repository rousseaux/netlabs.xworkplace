Il file AUTOEXEC.BAT Š un file batch speciale che viene, per definizione, eseguito all'avvio delle sessioni DOS.

Il suo scopo primario Š processare i comandi che definiscono il sistema per le sessioni DOS; permette inoltre di avviare altri file batch e programmi. In questo file si possono usare tutti i comadi DOS descritti in &os2; - Riferimento per i comandi (CMDREF.INF). I comandi pi— comunemente usati sono i seguenti: APPEND, CALL, ECHO, PATH, PROMPT, REM e SET.
