Un file programma Š un tipo speciale di &link_datafiles; che contiene codice eseguibile.

Il file programma di controllo (detto anche "driver") Š un file programma attraverso il quale &os2; controlla il funzionamento dei vari tipi di hardware.

E' un file eseguibile che non pu• essere avviato manualmente e che pu• essere caricato solo durante l'avvio del sistema. Fare riferimento alla documentazione del programma per dettagli ulteriori.
