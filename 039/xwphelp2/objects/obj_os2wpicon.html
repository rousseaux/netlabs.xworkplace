I file icona hanno generalmente .ICO come estensione del nome file e permettono di immagazzinare le icone &os2; sul disco.

Il file icona contiene bitmap in uno speciale formato che permette di definire trasparenze. Molti file icona contengono immagini in pi— risoluzioni, per evitare di dover scalare in caso di risoluzioni video differenti.

Come valore assunto i file icona sono associati con l'Editor di icone di &os2;. Per cambiare l'icona di un oggetto Š sufficiente trascinare un file icona nella pagina "Icona" del suo blocco appunti impostazioni.
