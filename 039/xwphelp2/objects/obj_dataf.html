I file dati vengono usati dai programmi per conservare su disco le informazioni. Tutti gli oggetti nel file-system che non sono file programma o &link_folders; vengono considerati da &os2; come file dati, indipendentemente dal fatto che contengano o meno dati immediatamente leggibili dall'utente.

I file dati generalmente sono di un tipo specifico. Si puï impostare in maniera esplicita il tipo di file dalla pagina "Tipo" nel blocco appunti impostazioni del file. Se al file non ä associato esplicitamente un tipo, &xwp; ne assegna automaticamente uno basandosi sulla estensione, cioä quella parte del nome file che segue l'ultimo punto.

Generalmente i file dati sono associati a &link_progobjs;. E' cosç sufficiente cliccare due volte sul file dati per far sç che si apra il programma associato con il file dati giÖ caricato, oppure selezionare un certo programma dal sottomenu "Aprire come" nel &popmenu;e del file dati.

Se al file dati non ä stato assegnato (nÇ esplicitamente nÇ automaticamente) un tipo &os2; lo associa con l'oggetto "Editor di testo", che di solito si trova nella cartella "UtilitÖ", a sua volta nella cartella "Programmi" sulla Scrivania. l'Editor di sistema.

Si possono configurare tipi di file ed associazioni di &xwp; dalla pagina "Tipi di file" dell'oggetto "Workplace Shell".

Si veda Tipi di file ed associazioni per una introduzione alle associazioni tra file di dati e programmi.

Si possono modificare le associazioni tra programmi e file usando la pagina "Associazioni" presente nel blocco appunti impostazioni di ogni programma.
