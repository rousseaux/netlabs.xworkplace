I file puntatore hanno generalmente .PTR come estensione del nome file e servono per immagazzinare su disco i puntatori mouse di &os2;.

Un file puntatore Š un particolare tipo di file icona, che contiene tra l'altro informazioni "hot spot" per il puntatore mouse di &os2;. I file puntatore permettono, per esempio, di cambiare forma al puntatore mouse dall'oggetto "Mouse" nella &link_syssetupfdr;.

I file puntatore possono anche essere usati come icone, mentre il contrario generalmente non Š possibile.

Come valore assunto i file puntatore sono associati con l'Editor di icone di &os2;.
