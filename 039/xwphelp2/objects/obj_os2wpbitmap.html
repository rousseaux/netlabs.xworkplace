I file bitmap hanno generalmente .BMP come estensione del nome file e permettono di immagazzinare immagini sui dischi. Il formato bitmap OS/2 consente di immagazzinare pi— versioni della stessa immagine all'interno di un solo file, se per esempio si vogliono supportare risoluzioni differenti.

L'installazione di &os2; comprende diverse immagini bitmap, che si possono usare come sfondo per le cartelle.

Come valore assunto i file bitmap sono associati all'Editor di icone di &os2;.
