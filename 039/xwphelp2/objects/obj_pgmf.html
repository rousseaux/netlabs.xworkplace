Il file programma Š un tipo particolare di file dati che contiene codice eseguibile.

I file dati con estensione del nome .EXE (che sta per "eseguibile") sono sempre considerati da &os2; come file programma. Se &xwp; trova all'interno del file i dati relativi ad un'icona, provvede ad assegnarla automaticamente al file programma. In caso contrario al file viene assegnata una icona standard, diversa a seconda del tipo di sessione.

In molti casi Š sufficiente cliccare due volte sul file programma per farlo partire. Tuttavia, se non si sa esattamente a cosa serve il programma, Š meglio stare attenti a quello che si fa, dato che molti file programmi non sono fatti per essere avviati manualmente.

Questo Š uno dei motivi per cui la maniera raccomandata di avviare le applicazioni &os2; Š creando degli appositi &link_progobjs;; la maggior parte dei pacchetti software comprende un programma di installazione che provvede a farlo automaticamente.
