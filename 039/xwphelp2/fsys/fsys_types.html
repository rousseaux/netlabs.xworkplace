Se dall'oggetto "Impostazioni &xwp;" si sono abilitate le cartelle turbo, &xwp; sostituisce la pagina "Tipo" nel blocco appunti impostazioni dei file dati con una nuova versione migliorata.

Nella nuova pagina tutti i tipi di file sono visualizzati in un albero, analogamente alla pagina "Tipi di file" nell'&link_wpsobj;. La differenza principale rispetto a quella pagina Š che qui per ogni tipo di file vi Š una casella selezionabile, che permette di impostare se il file dati ne fa parte o meno.

Si veda "Tipi e associazioni" per un'introduzione alle associazioni tra file di dati e programmi.

  • Selezionando una casella si attiva la relativa associazione. Per rimuovere l'associazione, deselezionare la casella.

  • Se la casella Š deselezionata, la relativa associazione Š disabilitata. Per attivarla, selezionare la casella.

  • Se la casella Š selezionata ma disabilitata significa che, per quel file, il tipo di file Š impostato automaticamente in base ad un filtro definito nell'oggetto "Workplace Shell". Se, per esempio, il tipo "Plain text" prevede in associazione il filtro *.txt e si ha un file dati il cui nome termina con .txt, questo file dati sar… automaticamente riconosciuto come file di tipo "Plain text".
