Il campo "Associazioni", in basso a destra, mostra le associazioni correnti.

In alto, in nero, sono visibili le associazioni definite per il tipo di file correntemente selezionato nel campo di sinistra; in colore differente (quello delle copie collegate sulla Scrivania) sono visualizzate le associazioni definite dai tipi di file superiori. In pratica, questo contenitore mostra tutte le opzioni presenti nel sottomenu "Aprire come" del file di dati del tipo corrente.

E' possibile trascinare all'interno del campo oggetti programma (gli oggetti delle classi WPProgram e WPProgramFile). E' Inoltre possibile riorganizzare l'ordine delle associazioni all'interno del contenitore trascinandole nella posizione desiderata (solo per gli elementi mostrati in nero, naturalmente).
