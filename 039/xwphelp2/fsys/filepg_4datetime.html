Il gruppo di campi "Data/Ora" raccoglie le informazioni relative alla data del file registrate da &os2;:
  1. Creazione mostra data e ora in cui il file Š stato creato. Questa informazione non viene mai modificata (a meno che il file non sia completamente riscritto da un'applicazione).

    Si noti che il file system FAT non conserva questa informazione, per cui questo campo non ha valore su una unit… FAT.

  2. Ultima modifica mostra data e ora in cui per l'ultima volta si Š scritto nel file. Tale valore sar… uguale o successivo alla data e ora di creazione.

  3. Ultimo accesso mostra data e ora in cui per l'ultima volta si Š letto dal file. Tale valore sar… uguale o successivo alla data e ora dell'ultima modifica.

    Si noti che il file system FAT non conserva neanche questa informazione, per cui anche questo campo non ha valore su una unit… FAT.
