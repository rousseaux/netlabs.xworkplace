&xwp;: File, pagina 1 &xwp;: pagina File &xwp; riunisce le tre pagine "File" normali in una nuova pagina "File" singola, che conserva le funzionalit… delle pagine "File" originali ma, per offrire una maggior praticit… d'uso, ne riunisce tutte le impostazioni.

Se si preferiscono le tre pagine "File" standard, disabilitare l'opzione relativa dalla pagina Funzioni che si trova nell'oggetto "Impostazioni &xwp;".

La pagina "File" Š presente solo nel blocco appunti impostazioni degli oggetti file system (&link_folders; e &link_datafiles;, fisicamente esistenti nel file system) e mostra le informazioni relative all'oggetto.

Esiste anche una seconda pagina "File" subito dopo questa, in cui sono visualizzati gli eventuali attributi estesi del file o della cartella.

Per una spiegazione dettagliata su ciascun campo, selezionare una voce del seguente elenco:

  • Nome fisico
  • Dimensione (byte)
  • Area di lavoro
  • Data/Ora
  • Attributi
  • Informazioni
