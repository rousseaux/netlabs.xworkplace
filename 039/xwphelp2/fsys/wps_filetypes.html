Questa pagina permette di configurare i nuovi tipi di file estesi di &xwp;, i loro filtri, e gli oggetti programma ad essi associati.

Si veda Tipi e associazioni per un'introduzione al concetto di associazione tra file dati e programmi.

La pagina consiste di tre sezioni, che permettono di configurare liberamente i tipi di file.

Per una spiegazione dettagliata di ogni campo, scegliere dalla lista sottostante:

  • Tipi di file
  • Filtri
  • Associazioni
  • Passa nell'ambiente il marcatore file dell'oggetto
Nota: nei &popmenu;i dei campi "Filtri" e "Associazioni" Š disponibile la voce "Importa filtri WPS", che permette di riutilizzare rapidamente i filtri WPS esistenti con i tipi di file.
