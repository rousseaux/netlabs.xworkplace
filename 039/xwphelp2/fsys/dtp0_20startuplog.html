Abilitando "Crea file WPSTART.LOG" &xwp; crea, nell'indirizzario radice dell'unit… di avvio, un file WPSTART.LOG che contiene il resoconto di quanto avvenuto durante l'avvio della WPS. Il file viene sovrascritto ad ogni avvio della Scrivania.

Attivare questa impostazioni Š molto utile se si sta cercando di individuare problemi che si verificano durante l'avvio di &xwp;.
