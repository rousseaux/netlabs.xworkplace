Il campo "Filtri", in alto a destra, mostra, per ogni tipo di file, i filtri definiti.

Tali filtri non sono in rapporto con i filtri predefiniti della WPS, che qui non vengono considerati e non possono essere modificati tramite questa pagina. Se i filtri estesi sono attivi &xwp; ignora completamente le associazioni standard della WPS.

Mentre la WPS usa i filtri per collegare al file dati un programma, &xwp; invece utilizza i filtri per assegnare ai file un tipo di file. Per esempio, definendo "*.htm" come filtro per i file di tipo "HTML", tutti i file che terminano per "*.htm" apparterranno automaticamente il tipo di file HTML (ed erediteranno le associazioni definite per quel tipo).

&xwp; permette comunque di importare i filtri WPS per poterli utilizzare con le associazioni estese. Cliccare col tasto destro sulla finestra "Filtri" e selezionare "Importare filtri WPS".
