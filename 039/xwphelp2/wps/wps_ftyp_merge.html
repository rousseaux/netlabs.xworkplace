La WPS associa i file ai programmi in accordo con il tipo di file e con i filtri: &xwp; migliora questo meccanismo di associazione basandolo solo sui tipi di file; questa finestra di dialogo permette di importare all'interno dei tipi di file di &xwp; i filtri precedentemente definiti come associazioni nella WPS.

Questa finestra Š non-modale, per cui Š possibile lavorare con la pagina di impostazioni "Tipi di file" anche se "Importa i filtri WPS" Š aperta. Selezionando un nuovo tipo di file, la finestra "Importare" si aggiorna automaticamente.

La finestra si chiude solo quando si preme il tasto "Chiudi".

Per importare tutte o alcune delle associazioni definite come filtri WPS nelle associazioni per tipo di file di &xwp;, procedere nella maniera seguente:

  1. selezionare o creare il tipo di file per cui si vuole importare il filtro, per esempio "Bitmap";

  2. nella finestra di dialogo "Importare" selezionare il filtro del quale si vogliono importare le associazioni, per esempio, *.BMP;

  3. il campo "Associazioni da aggiungere" mostrer… i programmi definiti con quel filtro. Selezionare le associazioni da aggiungere al tipo di file (nell'esempio il tipo "Bitmap"). Premere "Seleziona tutto" o "Deseleziona tutto" per selezionare o deselezionare in una volta sola tutti gli elementi presenti nel campo delle associazioni.

  4. Premere il pulsante "Collega" nella finestra di dialogo per collegare i programmi selezionati al tipo di file correntemente selezionato nella pagina impostazioni.
Questo procedimento pu• essere ripetuto quante volte si desideri. Si ricordi che &xwp; permette di definire tipi gerarchici per i file, in cui i tipi derivati ereditano le associazioni del tipo genitore.

Si veda Tipi e associazioni per un'introduzione alle associazioni tra file di dati e programmi.

Nota bene: questa finestra di dialogo non modifica i filtri della WPS. Se dall'oggetto "Impostazioni &xwp;" si disabilita l'associazione per tipo di file, i filtri WPS funzioneranno di nuovo come prima.
