Questa impostazione abilita i documenti predefiniti nelle cartelle.

Se questa impostazione Š selezionata i file di dati avranno una nuova voce nel loro &popmenu;e, che permetter… di definirli come "documento predefinito" per quella data cartella. Se nella cartella vi Š un file dati che Š stato dichiarato documento predefinito la cartella avr… una nuova voce nel suo &popmenu;e, usando la quale si potr… aprire tale documento senza dover aprire prima la cartella. In pratica, il documento si aggiunger… alle altre voci del menu "Aprire come".

Si veda la "Guida ad &xwp;" per una introduzione a questa funzione.

Modificando questa impostazione i &popmenu;i della cartella e del file di dati vengono modificati subito.
