Questa pagina permette di cambiare le impostazioni delle varie funzioni della &xshutdown; &xwp; Una introduzione a queste funzioni si trova nella Guida ad &xwp;. La modifica delle impostazioni ha effetto immediato.

Premere "Regredire" per ripristinare le impostazioni attive prima che la pagine fosse aperta.

Premere "Valore assunto" per ripristinare le impostazioni dell'installazione.

Per una spiegazione dettagliata di ogni campo, scegliere dalla seguente lista:

  • Svuota cestino
  • Conferma
  • Chiudi &warpcenter; prima della Scrivania
  • Chiusura automatica sessioni non-PM
  • Crea file XSHUTDWN.LOG
  • Crea cartella &xshutdown;
  • Far seguire riavvio
  • Animazione
  • Spegnimento PC
  • Salva file INI
