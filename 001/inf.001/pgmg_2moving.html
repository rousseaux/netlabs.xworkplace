Classes: XWPScreen
There are several ways to move between desktops in PageMage:
  • Click on the appropriate desktop in the PageMage window with the left mouse button.

  • You can use special global hotkeys for switching between desktops. Those are the cursor arrow keys plus any combination of the Ctrl, Alt and Shift modifiers.

    You can change the modifiers in the new "Screen" settings object, but the arrow keys are hard-coded into PageMage and cannot be changed.

    Those hotkeys work even if you have &xwp;'s global object hotkeys disabled. However, these hotkeys must be turned on first, which can be done in the "Screen" settings object as well.
