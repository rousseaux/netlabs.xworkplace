Bugfix release off the 1.0 branch and a few simple new features. "Bug #" numbers refer to the official XWorkplace bugtracker at http://xtracker.netlabs.org.

&xwp; 1.0.4 (March 13, 2005)

Bugfixes:

  • An annoying benign error box sometimes appeared complaining about sessions not being started in the foreground. The 'error' is now ignored. Bug 635. Fixed by Paul Ratcliffe.

    Disabling the XWP Hook with an active XCenter Window List widget caused endless traps. Bug 526. Fixed by Paul Ratcliffe.

    Fixed folder status bar colours which defaulted to Black on Grey rather than following the settings from the currently selected scheme. Fixed by Paul Ratcliffe.

New Features:

  • Objects created by the "Create New" function now have their entire title text selected, allowing easy overtyping. Added by Paul Ratcliffe.
