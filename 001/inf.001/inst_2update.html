XWorkplace can be installed over old versions of XFolder without any problems. WarpIN is capable of overwriting files which are currently in use (as the XFolder/XWorkplace files are by the WPS), so we have no problem there.

If you experience anything unexpected with your new version of XWorkplace, please look in the "Version information" section to find out what has been added to XFolder/XWorkplace.

Also, it is always a good idea to read the README file which comes with XWorkplace. You find that in the main XWorkplace installation directory.
