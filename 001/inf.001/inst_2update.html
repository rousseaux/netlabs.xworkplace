&xwp; can be installed over old versions of XFolder without any problems. WarpIN is capable of overwriting files which are currently in use (as the XFolder/&xwp; files are by the WPS), so we have no problem there.

If you experience anything unexpected with your new version of &xwp;, please look in the "Version information" section to find out what has been added to XFolder/&xwp;.

Also, it is always a good idea to read the README file which comes with &xwp;. You find that in the main &xwp; installation directory.
