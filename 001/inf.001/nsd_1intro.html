Are you too annoyed too by the fact that double-clicking on a URL or HTML file on your hard disk will always start a new instance of NETSCAPE.EXE, although Netscape is already running? This is a waste of time, memory, and system resources.

The Netscape DDE Interface is the solution. You can associate it with HTML and URL files, so that when you double-click on them, either Netscape will be started or, if it's already running, the HTML file or URL will simply be passed to the running instance.

NetscDDE works with both Navigator 2.02 and Communicator 4.04 for OS/2.
