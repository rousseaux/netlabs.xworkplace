Before installing &xwp;, please read the "Compatibility with other software" page in the appendices to this reference.

Starting with V0.9.0, &xwp; is installed using the WarpIN installer, a Netlabs project by Jens B„ckman and myself. This finally allows you to install &xwp; using a graphical interface and copy the files to a target directory.

Also, the way National Language Support (NLS) works has been reconsidered. &xwp; now per default comes with English NLS. If you wish to install a different language, download both the &xwp; base package and a second NLS package for your favorite language. Install &xwp; first, then the NLS package.

Then restart the WPS.

After the restart, &xwp; will automatically create a default configuration folder. If you wish to recreate this default configuration, go to the "Objects" page in the "&xwp; Setup" object and press the respective button.

XFolder neither changes CONFIG.SYS settings nor does it copy any files into &os2; system directories. Everything XFolder needs is situated in its own directory. The only things XFolder needs to work properly are a number of settings in OS2.INI, which will be removed when you uninstall XFolder.
