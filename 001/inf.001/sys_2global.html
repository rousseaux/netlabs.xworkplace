With the XWorkplace Global Settings, you can make XWorkplace behave differently. Here you will find settings which affect XWorkplace in general, such as its language support, as well as settings which will affect your computer as a whole. This is why they're called "global settings", as opposed to individual folder settings, which are explained on a separate page.

The major features of XWorkplace can be enabled or disabled in the new "XWorkplace Setup" object. Depending on the settings on the "Features" page, certain parts of XWorkplace will be completely disabled. This might be useful if you are only interested in certain features or if you have problems with others.
New "XWorkplace Setup" object

The detailed settings for the various XWorkplace features however are in the "Workplace Shell" object, which you will also find in your "System setup" folder.
New "Workplace Shell" object

Note that if you disable features in "XWorkplace Setup", some settings pages in "Workplace Shell" will not be shown if they don't make sense without a feature being enabled.

The different settings pages are not explained in detail here. You will find extensive help by pressing F1 or the "Help" button, which exists on every page.
