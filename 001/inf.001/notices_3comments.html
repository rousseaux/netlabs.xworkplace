&xwp; is an open source project with a development team.

The recommended way to contact all developers (and possibly other users as well) is to use the &xwp; mailing lists.

To contact the developers, send a message to one of the following addresses. You are also welcome to subscribe to any of these lists if you're interested in &xwp;'s progress.

  • xworkplace-user@yahoogroups.com is for &xwp; users. Use this address to submit bug reports if you don't understand the source code. Also, this is intended for public discussions about &xwp; (e.g. suggesting future features).

  • xworkplace-dev@yahoogroups.com is for messages related to the source code. This is used by the developers to communicate and can also be used by anyone else.
However, please read through the "Frequently Asked Questions" section before contacting the developers; maybe you will already find what you're looking for in there.

http://xworkplace.netlabs.org is &xwp;'s home page on the World Wide Web. Watch this space for news, updates, and public downloads.

A public bug list is also maintained on that homepage.

Reporting bugs. Please include the following information (the more, the better):

  • Please describe the situation in which the error occured and whether it is reproduceable.
  • Include the &os2; version you are running (Warp 3 or 4, including the fixpak level).
  • Include a list of other PM or WPS enhancers you are running.
  • Look in the root directory of your boot drive for a file named XFLDTRAP.LOG. This file logs &xwp;'s exceptions which might have occured on your system. If it exists, please include it in your message.
  • If you are running into problems with the eXtended Shutdown / Restart WPS features, please switch on the automatic log function on the "Shutdown" page of the Desktop's settings notebook. This will create a file named XSHUTDWN.LOG in the same location during shutdown. Please include this file also.
