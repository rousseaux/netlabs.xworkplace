XWorkplace is an open source project. As opposed to previous XFolder versions, the source code now entirely resides on the Netlabs server.

We use the Concurrent Versioning System (CVS) to cooperatively work on the source code. Basically, this means that there is a common source tree on an Internet server (in this case, the one of Netlabs), and several people commit their sources to that server.

To get the most current version of the XWorkplace source code, download the Netlabs Open Source Archive Client (NOSAC) from http:://www.netlabs.org/nosa, update your local "open source archive list" and check out the XWorkplace sources.

The NOSAC has been written by Christian Langanke and comes with extensive documentation.

In addition, a ZIP of the XWorkplace sources from which this XWorkplace version was built is available on the WWW page where you downloaded the XWorkplace archive from.
