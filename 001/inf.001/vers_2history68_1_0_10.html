Bugfix release off the 1.0 branch. "Bug #" numbers refer to the official XWorkplace bugtracker at http://xtracker.xworkplace.org.

&xwp; 1.0.10 (January 3, 2015)

The bugs below were fixed by these contributors: Paul Ratcliffe [pr], Alex Taylor [at], Steve Levine [sl].

Changes and Bugfixes:

  • Some records in Keyboard::Hot Keys were displayed incorrectly. Bug 249 [pr]

  • XWP beeps when dropping URL from Mozilla onto Desktop. Bug 336 [pr]

  • Various fixes for typos. in several places. Bugs 1201, 1202, 1205.

  • Font object sample view does not work with bitmap fonts. Bug 1204 [at]

  • All "Disk Details" pages show boot drive's stats. Bug 1206 [sl]

  • Xcenter widgets leak private memory. Bug 1210 [pr]

  • Xshutdown doesn't kill running programs in VIO windows. Bug 1216 [pr]

  • Support *.AFM and *.OTF font files. [at]

  • Prevent shutdown hang on DBCS systems. [at]
