Starting with V0.52, &xwp; is capable of supporting other languages than just English. This is called National Language Support (NLS). The &xwp; base package comes with English NLS.

For languages other than English however, you will need an &xwp; National Language Support (NLS) package. Such a package will contain a number of files, among them a new Dynamic Link Library (.DLL) for &xwp; itself, a new help file (.HLP) and a new User Guide book (.INF). Each of these files carries a three-digit language code in its respective file name, as defined in the &os2; Online Reference. For example, the English files, which are already included with &xwp;, all have a "001" code (for US English) in their names.

Please check http://&xwp;.netlabs.org for available NLS packages.

Note that &xwp; does not work with NLS packages written for XFolder V0.8x. Do not attempt to run this &xwp; version with those NLS packages. Since many language-dependent items have been rewritten, this will lead to really awkward behavior of your WPS and probably system crashes also.

NLS packages are installed using WarpIN too. If you wish to use a language other than English, after &xwp; has been installed, do not restart your computer yet, but install your NLS package next. Then restart the WPS.

When several languages are installed, the "&xwp; Setup" object offers you a "Language" drop-down box to change the current language.
