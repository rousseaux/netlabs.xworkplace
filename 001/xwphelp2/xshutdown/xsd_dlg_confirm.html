In the &xshutdown; confirmation window, which appears after selecting the "Shutdown" menu item, you can select if and how &xshutdown; will actually take place.

Push the "Shutdown" button to start &xshutdown; with the specified settings.

Press "Cancel" in order not to shut down at this time.

For a detailed explanation of each field, select from the list below:

  • Shut down only
  • Standard reboot
  • Reboot to...
  • Empty trash can
  • Show this dialog next time
