When you run a program in a full-screen session, the program covers the entire screen; you cannot see the windows and icons of other programs on your &os2; desktop, even though they remain open and keep running in the background. To see the windows and icons that are hidden, press Alt+Esc, or press Ctrl+Esc to display the Window List.

If you switch out of the full screen session while a program is running, the session is displayed iconized in the &xcenter;'s window list and in the &minwinv; With the icon you will see the name of the running program. With that icon, you can switch back to the full-screen session.

The icon in the &minwinv; also has a &popmenu; that you can use for closing the session or changing the session's properties. Note: You must use full screen instead of a window when a program

  • needs access to the physical screen selector,
  • directly works with the display adapter instead of using &os2;'s display drivers,
  • uses the &os2; monitor facility, or
  • attempts to register replacements for the Vio, Kbd, or Mou subsystems of the operating system.
