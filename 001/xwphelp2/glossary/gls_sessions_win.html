Windowed sessions have specific functions available that you can select from the system menu that is opened when you click on the title-bar icon of the window. Alternatively, click on the window with mouse button 2 to open a &popmenu;. Additional help is available for each menu choice.

With the menu, you can

  • change the font size of the characters that are displayed in the active window;
  • turn scrolling on or off;
  • mark text or graphics in the window for copying to the &os2; clipboard;
  • copy text or graphics that were marked from the window to the &os2; clipboard;
  • copy all text or graphics from the active window to the &os2; clipboard;
  • paste all text or graphics from the &os2; clipboard into the active window.
Note: You must use full screen instead of a window when a program
  • needs access to the physical screen selector,
  • directly works with the display adapter instead of using &os2;'s display drivers,
  • uses the &os2; monitor facility, or
  • attempts to register replacements for the Vio, Kbd, or Mou subsystems of the operating system.
