In the Workplace Shell, almost every object has a properties notebook associated with it where you can look at and possibly change the object's characteristics.

To open the properties notebook for an object, first click on the object with mouse button 2 to open its &popmenu;. Then select the "Properties" menu item.

You can also press the Alt+Enter keys as a keyboard shortcut to achieve this result.

To open the properties notebook for a folder, you can also open the folder's &popmenu; by clicking with mouse button 2 in an empty area of an open folder window, where you will find the "Properties" menu item as well. See folder views for details.

A window will open which usually has several pages in it, depending on the object's class. Each page in turn has a "Help" button where you can find out more about what the page does.
