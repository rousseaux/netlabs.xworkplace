If this setting is checked, instead of inserting the "Restart Desktop" item into the main &popmenu; of the Desktop, &xwp; will create a new "Shutdown" submenu containing menu items for the original &os2; shutdown, the eXtended shutdown of &xwp;, and the "Restart Desktop" feature.

This setting is disabled if &xshutdown; has not been enabled on the "Features" page in the "&xwp; Setup" object or if the Worker thread has been disabled on the "Paranoia" page in the same object.
