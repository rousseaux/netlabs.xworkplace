"Display status bar" will display a status bar in open views of this folder, depending for which views (Icon, Details, or Tree view) you have allowed status bars in the "Workplace Shell" object.

Changing this setting overrides the default setting specified on the "Status bars" pages in the "Workplace Shell" object.

This setting is disabled if status bars have been disabled altogether in the "&xwp; Setup" object. Also, this setting is always disabled for the Desktop.
