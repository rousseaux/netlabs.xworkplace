&xwp; replaces the default three "File" pages with a single new "File" page. This page has about the same functionality as the original "File" page, but combines all the settings into a single page for easier use.

If you'd like to get the original three "File" pages back, disable the respective setting on the "Features" page in the "&xwp; Setup" object.

The "File" page only exists in properties notebooks of file-system objects, that is, &link_folders; and &link_datafiles; (which physically exist in the file system), and shows you the object's file-system information.

Note that there's still a second "File" page behind this one. The &xwp; "File" page 2 however displays extended attribute information about a file or a folder.

For a detailed explanation of each field, select from the list below:

  • Real name
  • Size in bytes
  • Work area
  • Date/time
  • Attributes
  • Information
