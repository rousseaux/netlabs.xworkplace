This message appears when you have attempted to delete an object, but the &xwp; trash can could not be found.

This can have the following reasons:

  • The trash can was deleted for some reason. This is not easy to accomplish because the trash can protects itself against deletion, but still, it is possible.

  • The trash can has lost its object ID. To find its trash can, &xwp; looks for an object with the ID <XWP_TRASHCAN>.

  • Another object which is not the &xwp; trash can has the object ID <XWP_TRASHCAN>.
In any case, it is safe to press "Yes" to have a new trash can created on your Desktop.
