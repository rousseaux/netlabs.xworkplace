The "Information" fields allow you to specify additional information about this file in the file's extended attributes. Initially, all these fields are empty; you may set this to any information you like.
  • The "Subject" field can contain a brief summary of the purpose of this file. This is limited to 260 characters.

  • The "Comments" field can contain miscellanous notes on the file, for example restrictions, requirements, or any other information.

  • The "Keywords" field was designed to contain key text phrases which can identify the file in searches of, for example, database applications.
