Select "Refresh now" to make sure the folder view is up to date.

This will rescan the directory that corresponds to the folder, if necessary, to make sure the objects in the folder reflect the current file-system information. This operation might take a few seconds to finish. In addition, this completely repaints the folder view window.

Remarks:

  1. By default, &xwp; also defines the F5 folder hotkey as a shortcut to the "Refresh now" function.

  2. You can control the automatic refresh of folder views on the "Folder views" page in the &link_wpsobj;.
