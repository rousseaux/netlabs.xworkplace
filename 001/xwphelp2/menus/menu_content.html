With &xwp;'s folder content menus, you may take a look at the contents of a folder without actually having to open the folder. This saves you space on your display.

The folder content menus behave pretty similar to those of the &warpcenter;. If you click on a folder icon, a submenu will open to show you the folder's contents. If you click on another icon, the respective object will be opened in its default view (as if you had double-clicked on it in a regular folder). If you click on a folder menu with mouse button 2, the folder window will be opened (instead of the submenu).

Please consult the &xwp; User Guide to read more details about these functions and how to configure them.

To get rid of folder content menus in folder context menus, disable the respective setting on the "Context menus" pages in the "Workplace Shell" object and, for "favorite folders", on the "View" page in a folder's properties notebook.

To enable or disable the display of icons in folder content menus, use the respective setting on the aforementioned "Context menus" page.

To remove an object button from the &xcenter;, right-click on it and select "Remove".
