&xwp; allows you to add menu items globally to the &popmenu;s of all &link_folders; on the system. You have just encountered one of those variable menu items.

See Configuring &popmenu;s for instructions about how to change the default setup with the use of the special &cfgfdr;.

If you still have the default menu configuration which was set up during installation, the following new menu items are available: The standard content of the Desktop &popmenu; has the following variable menu items:

  • The "Command prompts" submenu contains items for opening command prompts. If you select one of the items in this submenu, a command prompt will be shown with the directory that corresponds to the current folder already changed to.

  • The "Create new" submenu contains items for creating new objects in the folder whose &popmenu; you have selected.

    See "Creating objects with Create New" for more.

  • The "Folder actions" submenu contains useful items for managing folders.

    "Show treesize" will start the Treesize program, which calculates the total space consumed by this folder and all of its subfolders on disk.

    "Pack tree (ZIP)" will compress the folder and all of the contained subfolders and files into a ZIP file. The zip file will be created in the parent folder of the current folder and carry the folder's name with a .ZIP extension. For example, if you select "Pack tree (ZIP)" for the C:\FOLDER\SUBFOLDER folder, this will create a C:\FOLDER\SUBFOLDER.ZIP file.

  • The "Quick Settings" submenu contains items with which you can quickly set certain folder properties without having to open its properties notebook.
  • The "&xwp;" submenu contains items for opening the &xwp; User Guide and the &cfgfdr;. Moreover, here you will find shortcuts to the new &link_wpsobj;, which contains many &xwp; global settings, and the new "&os2; Kernel" object also.
  • The "Treesize" menu item will start a program which will calculate the total disk space used by this folder and its subfolders. See "Treesize" for more explanations.
  • The "Netscape" menu item will start Netscape, having it display the directory of the folder whose &popmenu; you have selected.
