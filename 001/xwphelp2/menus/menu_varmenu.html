&xwp; allows you to globally add menu items to the context menus of all folders on the system. You have just encountered one of those variable menu items.

See Configuring context menus for instructions about how to change the default setup with the use of the special Configuration Folder.

If you still have the default menu configuration which was set up during installation, the following new menu items are available:

  • The "Command lines" submenu contains items for opening command line sessions. If you select one of the items in this submenu, a command line session will be opened with the directory of the folder already changed to.
  • The "Create another" submenu contains items for creating new objects in the folder whose context menu you have selected.

    See "Creating objects with &xwp;" for more.

  • The "Quick Settings" submenu contains items with which you can quickly set certain folder properties without having to open its properties notebook.
  • The "&xwp;" submenu contains items for opening the &xwp; User Guide and the Configuration Folder. Moreover, here you will find shortcuts to the new "Workplace Shell" object, which contains many &xwp; global settings, and the new "&os2; Kernel" object also.
  • The "Treesize" menu item will start a program which will calculate the total disk space used by this folder and its subfolders. See "Treesize" for more explanations.
  • The "Netscape" menu item will start Netscape, having it display the directory of the folder whose context menu you have selected.
