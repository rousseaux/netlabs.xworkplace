If extended sorting has been enabled in the "&xwp; Setup" object, &xwp; takes over sorting of your folders.

In the "Sort" folder menu, you can do the following:

  • You can sort the folder once by clicking on one of the sort criteria.

  • If you hold the "Shift" key down while clicking on one of the sort criteria, the folder's default sort criterion is changed, as if you had modified the setting on the "Sort" page in the folder's properties notebook.

  • If you select "Folders first", all subfolders are sorted to the top of the folder. See "Sort Criteria" for a detailed description.

  • If you select "Always maintain sort order", the folder will always be sorted according to the default sort criterion.
Unless you have specified sort settings for a specific folder, &xwp; will use the global sort settings from the "Sort" page in the "Workplace Shell" object.
