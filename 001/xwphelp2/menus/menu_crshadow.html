The "Create shadow" menu item will create new "shadow" objects that points to the currently selected objects and their contents.

A window will open that allows you to specify the folder where the shadows should be created. The window has several tabs to specify the target folder in various ways.

Remarks:

  1. Creating a shadow does not copy an object. Instead, a shadow is merely a pointer to the original object. See "About shadows" for details.

  2. If the target folder is already open, you can create the shadows more quickly by simply dragging and dropping the objects onto the target folder. Hold down the Ctrl and Shift keys while dragging to make sure the objects are not copied or moved.

  3. It is not recommended to create shadows for program files. Instead, create a program object.
