The "Title" field displays the name of the object. This name is displayed in folders together with the icon. Simply type in this field to change the object's title.

Your new title can be displayed on one or more lines below the icon. Press the Enter key as you type a new title to indicate the start of a new line. You can use any characters, including spaces, except the backslash (\).

If the object represents a physical file or a folder on a disk, changing the title will also change the object's file name ("real name"). Depending on the file system the object resides in, certain restrictions apply. See "Real names" for details.

Note that you can also quickly change the title of an object in a folder directly by holding down the "Alt" key and clicking on the object with mouse button 1. (This combination can be altered in the "Keyboard" settings object in your &syssetup; folder.)
