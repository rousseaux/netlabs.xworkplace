The "Screen" object lets you configure everything which concerns your screen display. Among other things, you can use this object to change your screen resolution and color depth.

This object has those notebook pages from the old "System" object which deal with the screen. Also, you can configure &xwp; &pgr; here.

Per default, this object can be found in the &syssetup; folder. Open the Desktop's &popmenu; and select "&syssetup;" to have that folder opened quickly.
