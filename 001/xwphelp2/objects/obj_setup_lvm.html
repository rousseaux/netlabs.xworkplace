The "Logical Volume Manager" (LVM) allows you to manage volumes on your system's hard disks.

This object resides in the &link_syssetupfdr;. Open the Desktop's &popmenu; and select "&syssetup;" to have that folder opened quickly.

The object will start the graphical version of LVM. &os2; also contains a text-mode version that you can start by typing LVM at a command prompt.

The &link_drivesfdr; also has a menu item called "Manage volumes" that will start the graphical version of LVM as well.
