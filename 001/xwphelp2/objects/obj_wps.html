The "Workplace Shell" object lets configure Workplace Shell settings in detail. This object has many notebook pages that deal with Desktop settings to fine-tune context menus, object and window views, confirmations, and much more.

This object has both the notebook pages from the old WPS "System" object that deal with the WPS, as well as new pages inserted by &xwp;. Please note that &xwp; only inserts certain pages into the "Workplace Shell" properties notebook if you have enabled the corresponding feature on the "Features" page of the "&xwp; Setup" object first. This applies, for example, to the "Sort" and "Hotkeys" pages.
