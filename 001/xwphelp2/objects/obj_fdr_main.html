Folders are objects that contain other objects. This makes them different from all other objects in the Workplace Shell. They can, for example, contain data files or other folders.

Every folder corresponds to a directory on disk. The nested directory structure on a disk is represented by a matching folder hierarchy.

If you double-click on a folder object, a folder view is opened, which displays all the objects that reside in the folder. There are three different folder views: Icon, Tree and Details view. To specifically which one is opened, instead of double-clicking on the folder, open its &popmenu; and select the view from the "Open" menu.
