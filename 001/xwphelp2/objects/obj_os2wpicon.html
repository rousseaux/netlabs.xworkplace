Icon files usually have a file name extension of .ICO and are used to store &os2; icons on disk.

An icon file contains bitmaps in a special format to allow for defining transparency. Most icon files contain pictures in several resolutions in order to avoid scaling under different device resolutions.

By default, icon files are associated with the &os2; Icon Editor. You can drag icon files onto the "Icon" page in any object's properties notebook in order to change the object's icon.
