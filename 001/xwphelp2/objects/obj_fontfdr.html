The "Fonts" folder is a special folder which shows you all fonts which are currently installed on your system.

This object resides in the &link_syssetupfdr;. Open the Desktop's &popmenu; and select "&syssetup;" to have that folder opened quickly.

It is recommended to always open the Fonts folder in details view, because only then can you see all the additional data the Fonts folder can display for your fonts. If the Fonts folder does not display font details correctly, set its "details class" to "Installed font" on the respective "View" page in the Fonts folder's properties notebook.

The Fonts folder has the following special features:

  1. When it is first opened, it will collect all fonts which are currently installed on your system and display them as font objects. Font objects have a special sample view which allows you to take a look at the font.

  2. You can quickly install fonts by dragging font files and dropping them into the open Fonts folder.

  3. You can quickly uninstall fonts by selecting "Uninstall" from a font object's &popmenu;.
Note: The Fonts folder is not intended to be used as a regular folder. Do not put regular objects into it. The only files you may drag into the Fonts folder are font files.
