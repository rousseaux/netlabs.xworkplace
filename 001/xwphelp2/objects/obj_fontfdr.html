The "&xwp; Font Folder" is a special folder which shows you all fonts which are currently installed on your system.

It is recommended to always open the font folder in details view, because only then you can see all the additional data the font folder can display for your fonts. If the font folder does not display font details correctly, set its "details class" to "Installed font" on the respective "View" page in the font folder's properties notebook.

The font folder has the following special features:

  1. When it is first opened, it will collect all fonts which are currently installed on your system and display them as font objects. Font objects have a special sample view which allows you to take a look at the font.

  2. You can quickly install fonts by dragging font files onto the open font folder.

  3. You can quickly deinstall fonts by selecting "Deinstall" from a font object's context menu.
Note: The font folder is not intended to be used as a regular folder. Do not put regular objects into it. The only files you may drag into the font folder are font files.
