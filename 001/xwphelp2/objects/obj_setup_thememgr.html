"Frame Control Themes" allows you to change the look of the frame controls, radio buttons, and check boxes in all windows on the system.

This object resides in the &link_lookfdr;.

Frame controls are the buttons at the top right of each window for closing, maximizing, or minimizing a window.

Note that making changes to these controls will require a system restart.
