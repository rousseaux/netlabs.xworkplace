The "Drives" folder is a special system folder created by &os2; which contains a list of all drives currently recognized by the system.

You can find the "Drives" folder in the &link_sysfdr; on your Desktop.

In this folder, each drive is represented by a separate object. This includes diskette drives, hard disk volumes, CD-ROM and DVD drives, and network resources currently assigned to a drive letter.

Double-click on the "Drives" folder icon to display the drive objects. To view the contents of a specific drive, double-click on the corresponding drive icon in the "Drives" folder.
