A network drive object represents a network resource that has been assigned a drive letter.

Double-click on the object to open a new window that displays the &link_folders; on the volume.

The "Local Network" folder on the Desktop allows you to access other network resources that do not currently have a drive letter. It also allows you to modify drive letter assignments for network resources. As long as you have free drive letters, you can use them to be able to access network resources more quickly.
