Double-click on the "Refresh Removeable Media" to have &os2;'s Logical Volume Manager rescan the system for removeable media that might have been inserted.

This is requivalant to typing LVM /REDISCOVERPRM at a command prompt.

For related information, select from the list below:

  • &link_drivesfdr;
  • Drives, volumes, and partitions
  • Formatting media
