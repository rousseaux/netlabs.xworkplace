The "&winos2setup;" object allows you to change the default settings for &winos2; programs.

This object resides in the &link_syssetupfdr;. Open the Desktop's &popmenu; and select "&syssetup;" to have that folder opened quickly.

This object allows you to, for example, enable sharing the clipboard between &winos2; sessions and the rest of the system, or elect that each &winos2; program should run in its own separate session.
