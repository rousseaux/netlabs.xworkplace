The "Current icon" field displays what icon (picture) is currently used to display the object in open folders.

Unless you have modified the object's icon, the default icon for the object is usually determined from the object's WPS class.

To modify the object's icon, do one of the following:

  • Drag and drop a new icon file (usually with an .ICO file name extension) into the "Current icon" field.

  • Push one of the buttons next to the icon display.
The following action buttons are available:
  • "Edit icon" will start the &os2; Icon Editor application with the current icon of the object. You can then modify the icon. If you save the icon with the Icon Editor, the object will pick up the change.

  • "Browse" will open a standard &os2; file dialog which allows you to find a new .ICO file to use for the object.

  • "Reset icon" will reset the object's icon to its default, which is determined by its WPS class. The button is disabled if the object is already using the class default icon.
Note that folder objects usually have a second "Icon" page after this page, which allows you to change the folder's animation icon.
