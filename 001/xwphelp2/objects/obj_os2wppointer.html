Pointer usually have a file name extension of .PTR and are used to store &os2; mouse pointer shapes on disk.

A pointer file is a special type of icon file that also contains "hot spot" information for the &os2; mouse pointer. You can use pointer files to change the system mouse pointer shapes in the "Mouse" object in the &link_syssetupfdr;, for example.

Pointer files can also be used as icons, but the reverse is usually not possible.

By default, pointer files are associated with the &os2; Icon Editor.
