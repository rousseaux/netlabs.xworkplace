This page allows you to specify the setup string for this setup string object and, optionally, an object on which this setup string will be invoked by default.

See "Setup string objects" for an introduction to setup string objects. See "Setup strings" for more information about WPS setup strings and a list of the most frequently used ones.

The "Setup string" entry field allows you to specify any valid WPS setup string. Always terminate each part of the setup string with a semicolon (";").

If "Confirm invocation" is checked, you will get a confirmation box when dropping another object onto the setup string object, that is, before the string is actually invoked on the object being dropped.

To have a default object for this setup string object (on which the setup string will be invoked by default when you double-click on the setup string object), drag any object to the container at the bottom. Press "Clear" to have that object removed from the setup string object.
