Double-click on a program object to launch the program associated with it.

A program object is really a kind of pointer to a program file. If you open the program object's properties notebook, you will see the path of the program file it points to.

Bei &os2; werden Programmobjekte zudem benutzt, um &link_datafiles; Anwendungen zuzuordnen. allows you to double-click on a data file and have a program open this data file automatically.

To create a new program object from a program file, do one of the following:

  • Open the &link_tempsfdr;, drag a new program object from the respective template, and fill in the data in the program's properties notebook.

  • Even more simply, drag the program file and drop it onto an open folder view (or the Desktop). This will automatically create a new program object for the program file with the program data already set.
