The &xwp; trash can contains all the objects that you have recently deleted. This allows you to recover objects that you deleted accidentally.

To delete objects into the trash can, do one of the following:

  • Drag and drop the objects onto the trash can. In that respect, the trash can operates similar to a regular folder.

  • Select the "Delete" menu item for a number of objects that are currently selected.

  • Press the "Delete" key on your keyboard to delete the objects that are currently selected.
The WPS "Delete" support can be enabled in the "&xwp; Setup" object.

If you open the trash can (it supports only the Icon and Details views), you can see all the objects that have been moved into it.

To restore a trash object, either open its context menu and select "Restore object", which will move this object back to its original location, or drag it out of the trash can into any open folder, which will restore the object in that location instead.

To permanently delete a trash object, select "Destroy object" from its context menu.

If the trash can is not empty, the trash can's context menu offers you an additional menu item called "Empty trash can", which will permanently delete all the objects which have been moved into the trash can.
