The &xwp; &cfgfdr; looks just like any folder, but is used by &xwp; to determine additional menu items for all folder &popmenu;s. All objects that you put into this folder will result in the creation of additional &popmenu; items.

This object resides in the &link_lookfdr;. Open the Desktop's &popmenu; and select "&syssetup;" to have that folder opened quickly. "XWorkplace Installation" folder.

See Configuring &popmenu;s for instructions about how to change the default setup.
