Enable this setting to add the "Refresh" menu to the main folder &popmenu; for open folder views. Otherwise, with OS/2 Warp 4 and later, the "Refresh" menu item is only in the "View" submenu, which many people do not find very convenient to reach.

Note also that, in the default configuration, the "F5" key has been defined as a hotkey to refresh the contents of the folder which is currently active.

This setting takes effect immediately.
