This setting only has an effect when "Show full path in folder title" is checked.
  • If this box is checked, &xwp; will append the full path of the folder to the existing title of the folder, enclosed in brackets. Example:

    &os2; System (C:\OS!2 System)

  • If this box is not checked, &xwp; will only display the full path of the folder. Example:

    C:\OS!2 System

This setting may be overridden on the "View" page in every folder's properties notebook.

Note: The folder title might be different from the last part of the "full path" display if the object's title is different from its real name. See "Real names" for details.
