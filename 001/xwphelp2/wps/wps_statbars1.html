On this page, you may configure the status bar which &xwp; can add to your open folder windows. Changing the settings will take effect immediately in the sense that all open folder windows will be updated instantly.

The "Enable folder status bars" setting determines the default visibility of status bars in open folder views. You can override this setting individually on the "View" page in each folder's properties notebook. This setting specifies whether status bars are visible if no individual folder setting has been defined.

To disable status bars completely, disable the respective item on the "Features" page in the "&xwp; Setup" object.

The settings in the "Visible in..." group determine globally for all folders in which folder views status bars can be displayed. These settings however only affect folders where status bars are visible, either because they have been enabled for a single folder in its properties notebook or because the global status bar setting is in effect for a folder.

The "Style" settings let you decide what the status bar will look like; just try out which style you like best, all open folder windows will be updated immediately if you change this setting.

Note that the status bars also accept fonts and colors being dropped on them from the respective palettes in the &syssetup; folder, which will update all status bars instantly also.

Press "Undo" to restore the settings that were in use before you opened this notebook page.

Press "Default" to restore the settings to the installation state.
