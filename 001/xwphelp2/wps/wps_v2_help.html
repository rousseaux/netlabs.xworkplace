Uncheck this box to remove the &os2; default "Help" &popmenu; entry, including the submenu which contains the different help options.

This setting affects all objects, including folders.

Even though this menu item can be removed, you can still press "F1" in folders to obtain help about the object that is currently selected.

Note that this will also remove the "&xwp; Product Information" menu item, which is normally placed into the "Help" submenu.
