If the "default documents" feature has been enabled, this will automatically make the "Folder's default document" item in a folder's "Open" context menu the default view for the folder if a default document exists.

In other words, if you declare a document a default document for its folder, you can then simply double-click on the folder to have the default document opened instead.

See the "&xwp; User Guide" for an introduction to this feature.

Changing this setting affects all folders immediately.
