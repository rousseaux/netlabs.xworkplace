On this notebook page you can set more global options for certain object menus on the system. However, while the options on the previous page affect all object menus, the items on this page affect folders, files, and disk objects only.

Press "Undo" to restore the settings that were in use before you opened this notebook page.

Press "Default" to restore the settings to the installation state.

For a detailed explanation of each field, select from the lists below.

  • Folder menu items affect all folder menus on the system. Note: "Sort" and "Arrange" have no effect on the Desktop because the Desktop has its own settings for these two menu items in its properties notebook.
    • "Paste"
    • "Find"
    • "Select" menu
    • "View" menu
    • Layout items
    • "Select by name"
    • "Sort" menu
    • "Arrange" menu
    • "Refresh now" in main menu
    • Add folder content menus
    • Display icons

    Note that more menu items may be added to folder menus according to the contents of the special &cfgfdr;. See Configuring &popmenu;s for details.

  • File menu items affect all file menus.
    • "Attributes" menu
    • "Copy filename"
  • Disk menu items affect disk objects in the "Drives" folder only.
    • "Check disk"
    • "Format disk"
