On this notebook page you can set global options for all object menus on the system.

Press "Undo" to restore the settings that were in use before you opened this notebook page.

Press "Default" to restore the settings to the installation state.

In the "Menu style" group, you can set whether the system should use long or short object menus. If you select "Short menus", all functions that can also be accessed via mouse or keyboard will be removed. You will see that the respective menu item settings will then be disabled in the group below.

(Note that you can override these two settings individually in each folder's properties notebook.)

If long menus are enabled, you can still determine in detail which menu items should be visible for object menus. Uncheck any setting in the "Common menu items" group to remove the respective menu item from all object menus on the system. Note that there are more selections on the following notebook page for items that are specific to certain objects.

For a detailed explanation of each field, select from the list below:

  • "Help"
  • "Create another"
  • "Copy"
  • "Move"
  • "Create shadow"
  • "Delete"
  • "Pickup"
  • "Lock in place"
  • "Print"
In addition you can set whether the system should display a menu bar with each open folder view. See Folder views for more.
