Selecting this setting will cause &xwp; to automatically open the properties notebook after it has created a new object from a folder &popmenu; template.

This applies only to templates in the &cfgfdr;. Dragging new objects from a template with the mouse will not be affected.

Note that even if you do not select this setting, you can still always have the settings opened if you hold down the Shift key while selecting the template menu item.
