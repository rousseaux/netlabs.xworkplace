If extended file types have been enabled in the "&xwp; Setup" object, &xwp; replaces the original "Associations" page in program properties notebooks with an enhanced version.

On the "Associations" page, you may set the file types that this program is associated with.

See Types and associations for an introduction to associations between data files and programs.

This page displays all file types in a tree, as they were defined on the "File types" page in the &link_wpsobj;. As opposed to that page, each file type on this page has a check box next to it which determines whether the program is currently associated with data files of that type.

  • If a box is checked, the respective association is currently set for the program. To remove the association, click onto the check box.

  • If a box is unchecked, the respective type association is not currently set for the program. Click onto the check box to add an association.
