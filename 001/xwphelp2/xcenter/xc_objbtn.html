The &xcenter; "object button" widget displays a button which represents any Desktop object.
  • If that Desktop object is a folder, you can click on the button and view the folder's contents.

  • If that Desktop object is not a folder, you can open the object in its default view by clicking on the button.
Object button widgets can easily be added to any position in the &xcenter; by dragging any Desktop object onto an empty area of the &xcenter; bar:

This will create an object button widget at the position where the Desktop object was dropped.
