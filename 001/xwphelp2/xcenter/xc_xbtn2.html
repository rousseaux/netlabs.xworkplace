In this dialog, you can selectively disable items in the menu that is displayed if you click on the &xbutton; widget. For example, if you would like to get rid of the "Run" dialog menu item, deselect "Run" in the "Visible menu items" group.

In addition, you can enter the file name of an OS/2 bitmap file that will be used for painting the &xbutton; widget itself. Use the "Browse" button to have a standard file dialog displayed, where you can find a bitmap file on disk.

Note: The &xbutton; supports only bitmap files in OS/2 1.3 or 2.0 formats. Do not attempt any other formats that might otherwise be supported by &os2;. Besides, the widget will resize itself according to the bitmap size, so use only small bitmaps.

If the entry field is empty (which is the default), or if the specified bitmap cannot be loaded, the standard &xwp; "X" "eCS" (eComStation) logo is displayed.
