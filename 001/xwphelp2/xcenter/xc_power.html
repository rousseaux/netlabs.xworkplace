The &xcenter; "Battery Power" widget will display the power state of your computer.

This widget only makes sense on laptops. On systems which have no battery installed, it will only report "No battery" and therefore be pretty useless. Besides, it requires that APM V1.1 be installed and working. As a rule of thumb, if the WPS "Power" object is working correctly on your laptop, the power widget should do also.

If there is a battery installed, the power widget will report two things:

  1. The current power source of the system, by displaying a small icon with either a plug (if the system is running on AC power) or a battery (if the system is currently using the battery).

  2. Next to the icon, it will show the current percentage of the battery load.
The display is updated every ten seconds only.

Note that you can drag and drop fonts and colors onto the power widget.

You can also double-click on the widget to have the regular WPS "Power" object opened quickly.
