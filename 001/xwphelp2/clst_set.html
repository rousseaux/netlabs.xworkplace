The "Class list" page in the "WPS Class List" object allows you to fine-tune that object's display.

If "Show all SOM classes" is checked, the class list will not only contain the regular WPS classes, but all SOM classes which are currently registered with the Workplace Shell process. This shows a lot lot more classes and will take significantly longer to display. Except for SOMObject, SOMClass, SOMClassMgr, Contained, and Container, all of these additional classes are undocumented.

This is probably only useful for curious SOM programmers. You will need to close the "WPS Class List" object and re-open it to view the change.

If "Show method information" is enabled, you'll get three windows in the open "WPS Class List" view. The third window displays all resolved methods for a selected class. If this is disabled, you'll get two windows only. This change takes effect immediately.
