The program object in the &cfgfdr; that corresponds to the menu item which you have selected has a "%**C" symbol in the parameter list. This means that the contents of the clipboard are supposed to be passed to the program as a parameter. However, the clipboard currently holds no text data: either it is empty, or it holds data of a different type (e.g. bitmap data).

Click on "Yes" to start the program anyway; the "%**C" symbol will be ignored, and no clipboard data will be passed to the program.

Click on "No" in order not to start the program.
