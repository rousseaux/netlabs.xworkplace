If object hotkeys are enabled in the "&xwp; Setup" object, &xwp; inserts the "Hotkeys" page into the "Keyboard" object's properties notebook.

This page allows you to view all currently defined object hotkeys at once and configure and remove hotkeys.

Note: Do not confuse object hotkeys with folder hotkeys, as displayed in the "Workplace Shell" object. Object hotkeys are used to open objects using a key definition and work everywhere, while folder hotkeys are shortcuts to certain folder commands and work in folders only.

The "Hotkeys" page lists all hotkeys which are currently defined on the system in a container which looks similar to a folder Details view.

By clicking on a hotkey item with mouse button 2, you will get a context menu which allows you to open the object's context menu, open the object's folder, or remove the hotkey altogether.

To change a hotkey, select "Object properties" from the context menu and switch to the "Icon" page in the object's properties notebook which has opened. This allows you to define a new object hotkey.
