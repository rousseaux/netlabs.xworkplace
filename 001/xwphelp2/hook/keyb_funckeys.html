This page allows you to register extra keys with the &xwp; PM hook which your keyboard might have and which are not supported by &os2; out of the box.

Before using this page, it is strongly recommended to consult the &xwp; User Guide for an introduction and some warnings. There is no error checking on this page with respect to registering dangerous function keys.

To register a new function key, right-click on the container background (or the column titles) and select "New function key". You'll get a dialog for this, press "Help" there for more directions.

After you have added a function key, you can use it as an object hotkey on the "Hotkeys" page and on the "Icon" pages in every object's properties notebook.

To edit an existing function key, right-click on the entry and select "Edit function key". This will show up the same dialog.

To remove a function key, right-click on it and select "Delete".
