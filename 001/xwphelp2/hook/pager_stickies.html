This PageMage notebook page allows you to define "sticky" windows, that is, windows which will be visible on all virtual desktops.

The page consists of a single container which lists all currently defined sticky windows. Initially, this container is empty.

See "Sticky windows criteria" to learn how PageMage uses the entries on this page.

To add a sticky window, right-click on an empty area of the window and select the menu item. A dialog pops up which allows you to define a new sticky window.

To remove a sticky window, right-click on the entry and select the menu item.

Press "Undo" to restore the settings that were in use before you opened this notebook page.

Press "Clear" to remove all sticky windows from the list.

Notes:

  • There is a maximum of 64 sticky window specifications.

  • Minimized windows are always sticky.
