If "Auto-hide mouse pointer" is enabled, &xwp; will automatically hide your mouse pointer from the screen if it hasn't been moved for the specified number of seconds. This might be helpful when reading text on the screen where the mouse pointer frequently gets into the way.

If "Do not hide if a menu is open" is also checked, the mouse pointer will not be hidden if a menu is currently open.

If "Do not hide if over a push button" is checked, the mouse pointer will not be hidden while over a push button.

Those two options might help you to locate the mouse pointer in places where clicking could be harmful.
