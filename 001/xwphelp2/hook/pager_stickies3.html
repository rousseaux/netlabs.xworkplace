This dialog allows you to specify a new "sticky window" definition. The dialog shows the switch list titles of all currently open windows to allow you to quickly add a new sticky window definition.

See "Sticky windows criteria" to learn how &pgr; uses the entries on the "Sticky windows" page.

To add a new sticky window, either type in the switch list title directly or select a switch list title from the list.

Press "OK" to have the item added to the list on the "Sticky windows" page.

Press "Cancel" in order to discard your selections.
