De "misc" directory bevat extra bestanden die nodig zijn voor de taalondesteuning
van XWorkplace. De makefile in deze directory zal alleen de bestanden kopi‰ren die
zijn gewijzigd naar XWPRUNNING, dus als u deze bestanden bijwerkt, doe het hier
in de bron en laat ze naar u XWorkplace installatie kopi‰ren door de makefile.

Dit zijn de bestanden:

crobj001.cmd        maakt de standaard configuratie map (REXX script)

instl001.cmd        maakt andere standaard installatie objecten (REXX script)

sound001.cmd        voegt nieuwe geluiden toe aan de INI systeem bestanden

xfldr001.tmf        berichten bestand (TMF) voor verschillende XWorkplace berichten.
                    Deze is nieuw sinds V0.9.0 en vervangt het OS/2 MSG bestand dat
                    eerder werd gebruikt. Dit is nu een plat tekst bestand.

drvrs001.txt        lijst van stuurprogramma's gebruikt door de "Stuurprogramma's"
                    pagina in "OS/2 Kernel".
                    Deze is nieuw sinds V0.9.0.

xfcls001.txt        WPS class beschrijvingen voor het "WPS Class List" object.
