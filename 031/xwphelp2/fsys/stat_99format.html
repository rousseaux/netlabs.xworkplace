Verschillende statusbalk codes die byte aantallen tonen vereisen een extra opmaak specificatie teken achter de eerste 2 code tekens. Bijvoorbeeld, met de $sX code (voor totale grootte van geselecteerde objecten), is X de opmaak specificatie, en moet ‚‚n van de volgende zijn:
  • b: Toon de waarde in bytes
  • k: Toon de waarde in kBytes, b.v. bytes / 1000
  • K: Toon de waarde in KBytes, b.v. bytes / 1024
  • m: Toon de waarde in mBytes, b.v. bytes / 1.000.000
  • M: Toon de waarde in MBytes, b.v. bytes / 1.048.576
  • a: Toon de waarde in bytes/kBytes/mBytes, afhankelijk van de grootte van de waarde
  • A: Toon de waarde in bytes/KBytes/MBytes, afhankelijk van de grootte van de waarde
