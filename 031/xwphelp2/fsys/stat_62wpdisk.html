Stationsobjecten (WPDisk class, b.v. de objecten in de &link_drivesfdr;) ondersteunen de volgende codes als uitbreiding op de codes voor alle objecten:
$fX
vrije ruimte op het station
$zX
totale ruimte op het station

In de bovenstaande twee situaties is X een speciaal opmaak specificatie teken.

$F
bestandssysteem type (FAT, HPFS, JFS, CDFS, ...)
$L
stationslabel

Opmerkingen:

De $f en $z codes zijn opnieuw gedefinieerd voor stationsobjecten. Normaal tonen zij de ruimte op het station waarop de map zich bevindt. Deze waarde is voor de "Stations" map altijd de zelfde als voor het station waarop de Werkplek zich bevindt. Zodoende is deze code geherdefineerd voor stationsobjecten.
