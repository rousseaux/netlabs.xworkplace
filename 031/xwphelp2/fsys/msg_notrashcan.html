Dit bericht wordt getoond wanneer u heeft geprobeerd een object te verwijderen, maar de &xwp; Prullenbak werd niet gevonden.

Dit kan de volgende redenen hebben:

  • De prullenbak is verwijderd. Dit is niet eenvoudig aangezien het prullenbakobject zichzelf tegen verwijderen beschermt maar toch is het mogelijk.

  • De prullenbak is zijn Object ID kwijt geraakt. Om de prullenbak te vinden, zoekt &xwp; naar een object met het ID <XWP_TRASHCAN>.

  • Een ander object dat niet de prullenbak van &xwp; is, gebruikt het object ID <XWP_TRASHCAN>.
Het is in ieder geval niet gevaarlijk om op "Ja" te drukken om een nieuwe prullenbak op uw werkplek te maken.
