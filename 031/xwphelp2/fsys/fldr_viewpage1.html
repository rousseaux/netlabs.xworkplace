Op deze notitieblok pagina kunt u, voor iedere map, een aantal weergave instellingen individueel configureren. Dit stelt u in staat om een aantal algemene instellingen, die zijn gespecificeerd in het Workplace Shell object, ongedaan te maken. Aanvankelijk zijn de instellingen hetzelfde als bij de algemene instellingen; als u hier iets wijzigt zal dat alleen voor deze map worden vastgelegd en hebben veranderingen in de algemene instellingen geen effect meer op deze map.

Wijzigingen gaan direct in.

  • Als een instelling is aangevinkt zal de betreffende instelling altijd voor deze map zijn ingeschakeld, onafhankelijk van wat u opgeeft in de algemene instellingen
  • Omgekeerd, als een instelling niet is aangevinkt, is de betreffende instelling altijd uitgeschakeld voor deze map.
Druk op "Beginwaarde" om de instellingen die in gebruik waren voor u deze map opende weer terug te zetten.

Druk op "Standaard" om deze map zich te laten gedragen overeenkomstig de algemene instellingen uit het Workplace Shell object. Dat zal de gegevens die voor deze map waren vastgelegd verwijderen.

Selecteer voor een gedetailleerde verklaring van elk veld uit de onderstaande lijst:

  • Favoriete map
  • Snel openen bij Werkplek start
  • Volledig pad in de titelbalk tonen
  • Op raster uitlijnen toevoegen
  • Reageer op map sneltoetsen
  • Statusbalk tonen
Extra informatie kan in het "Workplace Shell" object op de corresponderende helpschermen van de betreffende pagina's worden gevonden.
