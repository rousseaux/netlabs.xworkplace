In dit venster kunt u de object class selecteren waarvoor u de statusbalk informatie wilt instellen als slechts ‚‚n object in de map is geselecteerd.

Dit venster toont u alle ge‹nstalleerde WPS classes, om u een overzicht te geven van welke class wat is afgeleid.

Om echter de statusbalk codes te bepalen hoeft u alleen maar een class te selecteren die deze informatie kan leveren. Dergelijke classes worden in het zwart weergegeven, terwijl classes die dat niet ondersteunen in het grijs verschijnen en niet kunnen worden geselecteerd.

De boomstructuur is zodanig uitgebreid dat alle classes die de statusbalk informatie ondersteunen het eerste zichtbaar zijn.

De algemene statusbalk informatie wordt bepaald door de class XFldObject. Dan worden, als u een enkel object selecteert waarvoor geen class meer specifieke statusbalk informatie geeft (een voorbeeld hiervoor kan een printer object zijn), de codes van de XFldObject class gebruikt.

Meer specifieke sleutels worden echter bepaald door de classes: XFldDisk (voor "stations" objecten, zoals in de &link_drivesfdr;), WPProgram (voor onbenoemde programma objecten), WPFileSystem (voor mappen en gegevensbestanden, en WPUrl.

Merk op dat statusbalk informatie onafhankelijk is van de extra &xwp; classes. Dat betekent dat niet alleen de classes van &xwp; de statusbalk informatie ondersteunen, maar aan de andere kant ondersteunen ook niet alle &xwp; Classes dit even uitgebreid.
