• Om in een map een object te selecteren, klik op het object met muistoets 1.

  • Om de standaard kijk op een object te openen dubbelklik erop.

  • Om het &popmenu; van een opgeborgen object te openen, klik op het object met muistoets 2. Om het &popmenu; van de map zelf te openen, klik op een lege plek in de opbergplaats van de map.

  • Om een object te kopi‰ren of te verplaatsen of om er een reflectie van te maken, sleep en laat het los in een nieuwe map met muistoets 2.
Merk op dat als meerdere objecten zijn geselecteerd, de muis bewerking geldt voor alle geselecteerde objecten.
