• Om in een map een object te selecteren, klikt u op het object met muistoets 1.

  • Om de standaard inhoud van een object te openen, dubbelklik erop, zie: &xwp; dubbelklikken.

  • Om het &popmenu; van een object in een map te openen, klikt u op het object met muistoets 2. Om het &popmenu; van de map zelf te openen, klikt u op een lege plek in de opbergplaats van de map.

  • Om een object te kopi‰ren, te verplaatsen of om er een reflectie van te maken, zie: Reflecties , kunt u het object slepen en loslaten in een nieuwe map met muistoets 2 ingedrukt.
Merk op dat als meerdere objecten zijn geselecteerd, zie Selecteren, de muisbewerking geldt voor alle geselecteerde objecten.
