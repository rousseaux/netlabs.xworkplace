In een "Local Area Network" zie: LAN introductie, is een server een computer die zijn hulpbronnen deelt met andere computers, de clients.

Typisch in LAN's is dat een server een krachtige computer is met een besturingssysteem met uitgebreide functies speciaal ontworpen om een groot aantal netwerkvragen en meerdere gebruikers af te handelen, ieder met aparte toegangsrechten. De client computers kunnen geen hulpbronnen delen maar alleen de hulpbronnen op de server benaderen.

Daar tegenover staat dat in een peer netwerk iedere computer als een server of client of beide kan optreden.

Gebruik de &link_wpnetgrp; om alle servers te bekijken die hulpbronnen delen op het LAN.
