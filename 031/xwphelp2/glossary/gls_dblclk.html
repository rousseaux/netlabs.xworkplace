"Dubbelklikken" op een object betekent tweemaal klikken met muistoets 1 terwijl de muisaanwijzer op het object staat.

Dubbelklik op een object om het op zijn standaard wijze te bekijken. Als u dubbelklikt op een map zie Mappen, opent zich een &xwp;: Map weergaven die de objecten laat zien die in de map zitten.

Merk op dat u de dubbelklik snelheid kunt instellen (dat is, de hoeveelheid tijd die mag passeren tussen twee klikken alvorens &os2; niet langer twee klikken beschouwt als een dubbelklik) in het "Muis" object in de &link_syssetupfdr;.
