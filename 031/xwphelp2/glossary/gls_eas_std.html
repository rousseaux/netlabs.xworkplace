&os2; stelt zelf een aantal &xwp;: Uitgebreide kenmerken (EA's) vast. Deze worden Uitgebreide standaard kenmerken genoemd. De veelvuldig gebruikte Uitgebreide standaard kenmerken zijn:
  • .CLASSINFO wordt door de Workplace Shell gebruikt om zijn object specifieke gegevens met bestandssysteem objecten op te slaan. Dit zijn binaire gegevens die zaken bevatten als de naam van de class van het object en de meeste van de instellingen als de standaardwaarden in het Instellingen notitieblok van de objecten zijn gewijzigd.

  • .COMMENTS .KEYWORDS, en .SUBJECT worden gebruikt om extra informatie over het bestandssysteem object op te slaan en komen overeen met de respectievelijke velden op de "Bestand" pagina van het instellingen notitieblok van de objecten.

  • .ICON wordt gebruikt als een bestandssysteem object een niet-standaard pictogram wordt toegewezen dat dan ook in de Workplace Shell zal worden getoond. Als u het pictogram, op de Pictogram pagina, van een instellingen notitieblok van een bestandssysteem object wijzigt, wordt dit pictogram opgeslagen in deze EA. (Als toevoeging, bij mappen bevat .ICON1 het open map pictogram van die map.)

  • .LONGNAME wordt door de Workplace Shell gebruikt om de naam van een object in op te slaan als deze niet overeenkomt met de naam conventies van het bestandssysteem waar het object toe behoort. Zie:: Echte namen voor details.

  • .TYPE bevat het bestandstype van het object als de standaardwaarde, in het instellingen notitieblok, is gewijzigd.
