De onderdelen in het "Maak een andere" menu zullen een nieuw object van het zelfde type maken als het momenteel geselecteerde object. Echter, het huidige object zal niet gekopieerd worden. In plaats daarvan zal er een nieuw object gemaakt worden van het zelfde type maar met standaard instellingen en gegevens.

Het "Maak een andere" menu heeft een standaard menu keuze. Alleen als u de pijl rechts van "Maak een andere" kiest zal er een submenu getoond worden. Dit menu bevat een lijst van alle modellen op het systeem die van het zelfde type zijn als het huidige object. Echter, alhoewel deze modellen van het zelfde type zijn, kunnen ze toch andere instellingen en gegevens bevatten dan het huidige object.

Het resultaat van het gebruik van "Maak een andere" is hetzelfde als het aanmaken van een object op basis van een model.

Let op: Het "Maak een andere" menu laat objecten zien van het zelfde type als het huidige object. Daar tegenover staat dat het "Maak een nieuwe" menu altijd dezelfde lijst met modellen laat zien, die gebaseerd is op de &xwp; &cfgfdr;. Standaard staan veel gebruikte object types zoals mappen in dat menu, maar u kunt die lijst zelf aanpassen.
