De onderdelen in het "Nieuw object" menu zullen een nieuw object van hetzelfde type maken als het momenteel geselecteerde object. Het huidige object echter zal niet gekopieerd worden. In plaats daarvan zal er een nieuw object gemaakt worden van hetzelfde type maar met standaard instellingen en gegevens.

Het "Nieuw object" menu heeft een standaard menukeuze. Alleen als u de pijl rechts naast "Nieuw object" kiest zal er een submenu getoond worden. Dit menu bevat een lijst van alle modellen op het systeem die van hetzelfde type zijn als het huidige object. Alhoewel deze modellen van hetzelfde type zijn, kunnen ze toch andere instellingen en gegevens bevatten dan het huidige object.

Het resultaat van het gebruik van "Nieuw object" is hetzelfde als het maken van een object op basis van een model.

Let op: Het "Nieuw object" menu laat objecten zien van hetzelfde type als het huidige object. Daar tegenover staat dat het "Nieuw object" menu altijd dezelfde lijst met modellen laat zien, die gebaseerd is op &xwp; de &cfgfdr; van &xwp;. Standaard staan veel gebruikte objecttypen zoals mappen in dat menu, maar u kunt die lijst zelf aanpassen.
