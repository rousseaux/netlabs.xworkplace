&xwp; biedt u de mogelijkheid om globaal menu onderdelen toe te voegen aan de &popmenu;'s van alle &link_folders; op het systeem met het gebruik van een speciale &cfgfdr;, die er net zo uitziet als een normale WPS map.

Deze map staat in uw &link_lookfdr;. "XWorkplace Installatie" map.

Wat u ook in die map stopt zal direct tevoorschijn komen in de &popmenu;'s van al uw mappen op uw systeem. Let echter op dat bepaalde object classes een speciale betekenis hebben in de context van de &cfgfdr;:

  1. Programma objecten in de &cfgfdr; zullen gestart worden als het betreffende menu onderdeel wordt geselecteerd. (Eigenlijk wordt niet het programma object gestart, maar het programma dat gedefinieerd is in het programma object zijn instellingen notitieblok.)

    &xwp; kan een aantal handige trucjes uithalen met programma objecten. Zie "Extra mogelijkheden met programma objecten" voor meer informatie.

  2. Door het plaatsen van een andere map in de &cfgfdr; zal er een trapsgewijs submenu aangemaakt worden in de &popmenu;'s. In deze submap kunt u meer objecten en mappen plaatsen. Op deze manier kunt u een hele &popmenu; hi‰rachie maken (zoals u kunt zien in de standaard opzet). Er is geen limiet op het nesten van mappen, dus u zou hi‰rachie‰n kunnen maken die vele niveaus diep zijn.
  3. Een model of reflectie van een model zal resulteren in het aanmaken van een nieuw object in de map waarvan het &popmenu; gebruikt is.

    Bijvoorbeeld, als u een map model aangemaakt heeft met speciale instellingen voor lettertypes, kleuren enz., dan kunt u een reflectie daarvan opnemen in een Configuratie Map, en als u zijn overeenkomstige menu onderdeel selecteert vanuit een map zijn &popmenu;, zal er een nieuwe map gemaakt worden in deze map, net alsof u een map gesleept had vanaf een model.

    Zie "Aanmaken objecten met Aanmaken Nieuw" voor meer.

  4. Een instellingen tekst object zal de instellingen tekst uitvoeren, die opgeslagen is in het instellingen tekst object, op de map waarvan het &popmenu; onderdeel is geselecteerd.
  5. Alle andere objecten zullen simpelweg geopend worden als hun betreffende menu onderdeel geselecteerd wordt. Dit is inclusief reflecties van programma objecten, reflecties van mappen, "echte" bestanden, reflecties daarvan, en ga zo maar door.

    Bijvoorbeeld, u kunt een reflectie van uw favoriete gegevensbestand (b.v. "Te-doen lijst") opnemen in de &popmenu;'s.

  6. Als een speciale mogelijkheid kunt u menu scheidingstekens toevoegen door een programma object met de exacte naam van "---" (drie koppeltekens) op te nemen in een &cfgfdr;. Zie wederom "Extra mogelijkheden met programma objecten" voor details.
Let op dat het "~" karakter binnen een object zijn naam veranderd in een onderstreept karakter in het &popmenu;, om toetsenbord sneltoetsen mogelijk te maken.
