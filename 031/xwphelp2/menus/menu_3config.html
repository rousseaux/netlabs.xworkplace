&xwp; biedt u de mogelijkheid om algemene menuonderdelen toe te voegen aan de &popmenu;'s van alle &link_folders; op het systeem met het gebruik van een speciale &cfgfdr;, die er net zo uitziet als een normale WPS map.

Deze map zit in de &link_lookfdr; van de &syssetup; map. &xwp;: Installatie map.

Wat u ook in deze map stopt het zal direct te zien zijn in de &popmenu;'s van al uw mappen op uw systeem. Let er echter op dat bepaalde object classes een speciale betekenis hebben in de context van de &cfgfdr;:

  1. Programma objecten in de &cfgfdr; zullen worden gestart als het betreffende menu onderdeel wordt geselecteerd. (Eigenlijk wordt niet het programma object gestart, maar het programma dat gedefinieerd is in het instellingen notitieblok ven het programma object.)

    &xwp; kan een aantal handige trucjes uithalen met programma objecten. Zie: Extra mogelijkheden met programma objecte voor meer informatie.

  2. Door het plaatsen van een andere map in de &cfgfdr; zal er trapsgewijs een submenu worden gemaakt in de &popmenu;'s. In deze submap kunt u meer objecten en mappen plaatsen. Op deze wijze kunt u een hele &popmenu; hi‰rachie maken (zoals u kunt zien in de standaard opzet). Er is geen limiet aan het nesten van mappen, dus u zou hi‰rachie‰n kunnen maken die vele niveaus diep zijn.
  3. Een model of de reflectie van een model zal resulteren in het maken van een nieuw object in de map waarvan het &popmenu; in gebruik is.

    Bijvoorbeeld, als u een map model gemaakt heeft met speciale instellingen voor lettertypen, kleuren enz., dan kunt u een reflectie daarvan opnemen in een Configuratie Map, en als u het overeenkomstige menu onderdeel selecteert vanuit het &popmenu; van een map, zal er, in deze map, een nieuwe map gemaakt worden, net alsof u een map gesleept had vanaf een model.

    Zie: Maken van objecten met "Nieuw object voor meer informatie.

  4. Een Object instellingen reeks zal de instellingen tekst uitvoeren, die opgeslagen is in de object instellingen reeks, op de map waarvan het &popmenu; onderdeel is geselecteerd.
  5. Alle andere objecten zullen simpelweg geopend worden als hun betreffende menu onderdeel geselecteerd wordt. Dit is inclusief reflecties van programma objecten, reflecties van mappen, "echte" bestanden, reflecties daarvan, en ga zo maar door.

    Bijvoorbeeld, u kunt een reflectie van uw favoriete gegevensbestand (b.v. "Te-doen lijst") opnemen in de &popmenu;'s.

  6. Als een speciale mogelijkheid kunt u menu scheidingstekens toevoegen door een programma object met de exacte naam van "---" (drie koppeltekens) op te nemen in een &cfgfdr;. Zie wederom: Extra mogelijkheden met programma objecten voor details.
Let op dat, om het gebruik van toetsenbord sneltoetsen mogelijk te maken, het "~" karakter binnen de naam van een object verandert in een onderstreept karakter in het &popmenu;.
