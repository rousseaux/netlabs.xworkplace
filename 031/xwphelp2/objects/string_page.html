Deze pagina geeft u de mogelijkheid om de instellingen reeks op te geven voor dit instellingen reeks object en, als extra, een object waarop deze instellingen tekst standaard toegepast wordt.

Zie: "Object instellingen reeksen" voor een introductie in object instellingen reeksen. Zie: "Instellingen reeksen" voor meer informatie over Werkplek instellingen reeksen en een lijst van de meest frequent gebruikte.

Het "Instellingen reeks" invoerveld biedt u de mogelijkheid om elke geldige Desktop instellingen reeks op te geven. Sluit altijd elk deel van de instellingen reeks af met een puntkomma (";").

Als "Bevestig implementatie" geselecteerd is, krijgt u een bevestigingskeuze als u een ander object op de object instellingen reeks laat vallen, dat wil zeggen, voordat de reeks werkelijk toegepast wordt op het object dat u erop heeft laten vallen.

Om een standaard object te hebben voor deze object instellingen reeks (waarop de instellingen tekst standaard van toepassing zal zijn als u dubbelklikt op de object instellingen reeks), sleept u elk willekeurig object naar het veld onderaan. Druk op "Schoon" om dat object te verwijderen uit het instellingen tekst object.
