Deze pagina geeft u de mogelijkheid om de instellingen tekst op te geven, voor dit instellingen tekst object en, optioneel, een object waarop deze instellingen tekst standaard toegepast wordt.

Zie "Instellingen tekst objecten" voor een introductie in instellingen tekst objecten. Zie "Instellingen teksten" voor meer informatie over Desktop instellingen teksten en een lijst van de meest frequent gebruikte.

Het "Instellingen tekst" invoerveld biedt u de mogelijkheid om elke geldige Desktop instellingen tekst op te geven. Sluit altijd elk deel van de instellingen tekst af met een puntkomma (";").

Als "Bevestig implementatie" geselecteerd is, krijgt u een bevestigingskeuze als u een ander object op het instellingen tekst object laat vallen, dat wil zeggen, voordat de tekst werkelijk toegepast wordt op het object dat u erop heeft laten vallen.

Om een standaard object te hebben voor dit instellingen tekst object (waarop de instellingen tekst standaard van toepassing zal zijn als u dubbel klikt op het instellingen tekst object), sleept u elk willekeurig object naar de container onderaan. Druk op "Schoon" om dat object te verwijderen uit het instellingen tekst object.
