WPFolder is de WPS class voor alle mappen (zie WPS classes voor details). In aanvulling op de instellingen teksten die ondersteund worden door de WPObject class, ondersteund WPFolder de volgende teksten:
ACCELERATORS={YES|NO|DEFAULT} (&xwp; alleen)
Bepaald voor een individuele map of die reageert op map sneltoetsen of niet. Deze instelling komt overeen met die op de "Beeld" pagina in de map zijn instellingen notitieblok. YES of NO zal deze instelling bepalen voor deze map ongeacht wat opgegeven is in de globale instellingen, terwijl DEFAULT dit zal aanpassen aan de globale waarde.

ALWAYSSORT={YES|NO|DEFAULT}
Bepaald voor een individuele map of die altijd de sorteervolgorde zal vasthouden of niet. Het openen en toevoegen van een object aan deze map kan langer duren als de sorteervolgorde bijgehouden moet worden. Denk er om dat de DEFAULT optie alleen beschikbaar is bij &xwp;, wat de instelling zal herstellen naar hetgeen u gekozen hebt op de "Sorteer" pagina in het "Workplace Shell" object.

BACKGROUND=naam,mode,factor,bgtype,bgcol
Dit bepaald de achtergrond van de map, waarbij:

naam = De bestandsnaam van het plaatje. Deze naam dient volledige gekwalificeerd te worden met het pad van het plaatjes bestand. "?:\" is toegestaan om de boot schijf aan te geven.

mode = Plaatjes mode. Deze mode kan ‚‚n van de volgende zijn:
N = Normaal plaatje
T = Getegeld plaatje
S = Geschaald plaatje

factor = Schaal factor.

bgtype = Achtergrondtype. Dit kan een van de volgende zijn:
I = Plaatje
C = Alleen kleur

bgcol = Achtergrond kleur. Deze kleur heeft 3 nummers die RGB waarden vertegenwoordigen, afgescheiden door spaties (r g b, voor rood, groen, en blauw waarden).

Bijvoorbeeld: BACKGROUND=C:\OS2\BITMAP\OS2LOGO.BMP,S,3,I of BACKGROUND=(none),,,C,0 128 128

DEFAULTSORT=crit (&xwp; alleen)
Dit zal de standaard sorteer criteria aanpassen voor een individuele map naar iets anders dan de globale sorteer criteria. crit mag een van de volgende zijn:
-1 sorteer op type
-2 sorteer op naam
-3 sorteer op object class
-4 sorteer op bestandsnaam uitbreiding
255 herstel naar globale standaard sorteer criteria

Denk er om dat afhankelijk van de map sorteer class, waarden tussen 0 en 100 ook ondersteund kunnen worden. Die waarden komen overeen met indices voor object detail kolommen in Details overzicht en zijn class specifiek. Experimenteer met de instellingen tekst waarden die genoemd worden in een map zijn instellingen notitieblok.

DEFAULTVIEW=DEFAULT
Stelt het standaard open overzicht in naar wat opgegeven is in het "Workplace Shell" object. Dit is de standaard waarde.

Let er op dat DEFAULTVIEW werkelijk een WPObject instellingen tekst is, maar de betekenis van de overzicht instelling heeft alleen betekenis voor WPOBject subclasses, zoals WPFolder.

DEFAULTVIEW={SETTINGS|ICON|TREE|DETAILS}
Stelt het standaard open overzicht in voor het specifieke map overzicht.

Let er op dat DEFAULTVIEW werkelijk een WPObject instellingen tekst is, maar de betekenis van de overzicht instelling heeft alleen betekenis voor WPObject subclasses, zoals de WPFolder.

DETAILSCLASS=classname
Dit stelt de object class in voor welke de details getoond worden in het details overzicht. De standaard object class is WPFileSystem. U kunt hier beter niet mee rommelen.

DETAILSFONT=grootte.facename
Wijzig het lettertype dat gebruikt wordt voor het details overzicht. Bijvoorbeeld: 9.WarpSans Bold. De standaard waarde is 8.Helv (op Warp 3)

DETAILSSHADOWCOLOR=kleur (alleen &warp4; en hoger)
Tekst kleur voor een reflectie object in het Details overzicht. kleur mag een standaard kleur naam zijn of een r g b set van waarden (voor rood, groen, blauw waarden).

DETAILSTEXTCOLOR=kleur (alleen &warp4; en hoger)
Tekst kleur voor normale objecten in het Details Overzicht. kleur mag een standaard kleur naam zijn of een r g b set van waarden (voor rood, groen, blauw waarden).

DETAILSVIEW=stijl
Dit zet details overzicht naar de opgegeven stijl. Mogelijke waarden zijn:
MINI kleine pictogrammen; dit is de standaard waarde.
NORMAL gewone grootte pictogrammen.

FAVORITEFOLDER={YES|NO} (&xwp; alleen)
Stelt voor een individuele map in of zijn inhoud bekeken kan worden vanuit alle &popmenu;'s op uw systeem. Deze instelling komt overeen met degene op de "Beeld" pagina in de map zijn instellingen notitieblok. Standaard is NO. Er is geen globale waarde voor deze instelling.

FULLPATH={YES|NO|DEFAULT} (&xwp; alleen)
Stelt voor een individuele map in of zijn volledige pad wel of niet getoond wordt in zijn venstertitel. Deze instelling komt overeen met degene op de "Beeld" pagina in de map zijn instellingen notitieblok. YES of NO zal deze instelling voor deze map ongeacht wat er ingesteld is in de globale instelling, terwijl DEFAULT de instelling op de globale waarde zal zetten.

ICONFONT=grootte.facename
Instellingen tekst die gebruikt wordt om het lettertype van een pictogram overzicht aan te passen. Bij voorbeeld: 10.Helv

ICONNFILE=index,bestandsnaam
Dit stelt de bestandsnaam voor het bewegende (open map) pictogram in. De index waarde moet op 1 gezet worden. Het standaard bestand (".ICO") bevat de map zijn open map pictogram.

ICONNRESOURCE=index,id,dllfile
Stelt het bewegende (open map) pictogram in op gegevens in het bronbestand. De index waarde moet op 1 gezet worden. De id is de identiteit van een pictogram bron in de DLL dllfile. De opgegeven bron is de map zijn open map pictogram.

ICONSHADOWCOLOR=kleur (alleen &warp4; en hoger)
Tekst kleur voor reflectie pictogrammen in het Pictogram overzicht. kleur mag elke standaard kleur naam zijn of een r g b set van waarden (voor rood, groen, blauw waarden).

ICONTEXTBACKGROUNDCOLOR=kleur (alleen &warp4; en hoger)
Achtergrond kleur voor normale tekst in de Pictogram, Boomstructuur en Details overzicht. kleur mag een standaard kleur naam zijn of een r g b set van waarden (voor rood, groen, blauw waarden).

ICONTEXTCOLOR=kleur (alleen &warp4; en hoger)
Tekst kleur voor normale pictogrammen in het Pictogrammen overzicht. kleur mag een van de standaard kleuren namen zijn of een r g b set van waarden (voor rood, groen, blauw waarden).

ICONTEXTVISIBLE={YES|NO}
De namen van objecten in een map worden wel of niet getoond in een Pictogrammen Overzicht.

ICONVIEW=stijl1[,stijl2...]
Dit stelt het pictogram overzicht in naar de opgegeven stijlen. Mogelijke waarden zijn: FLOWED NOGRID INVISIBLE NONFLOWED MINI NORMAL
De standaard waarde is NOGRID,NORMAL
Deze instellingen komen overeen met de "Pictogram overzicht" pagina in een map zijn instellingen notitieblok.

ICONVIEWPOS=x,y,cx,cy
Dit stelt de initi‰le pictogram overzicht positie en grootte van de map in op het scherm. Deze waarden zijn percentages (0-100) van de schermgrootte.

MENUBAR={YES|NO} (alleen &warp4; en hoger)
De menubalk van de map wordt getoond of niet.

QUICKOPEN={YES|NO|IMMEDIATE} (&xwp; alleen)
Bepaald voor een individuele map of die wel of niet vooraf opgebouwd wordt bij het opstarten van de Werkplek. Deze instelling komt overeen met degene in de "Beeld" pagina in de map zijn instellingen notitieblok. YES of NO bepaald de instelling voor deze map. Standaard is NO. Er is geen globale waarde voor deze instelling.

Als een speciale opdracht, QUICKOPEN=IMMEDIATE zal de map direct opgebouwd worden. Dit kan gebruikt worden in een REXX script om de opbouw van een map te forceren zonder het te hoeven openen. Als gevolg daarvan is QUICKOPEN=IMMEDIATE geen instelling, maar meer een opdracht dat uitgevoerd wordt op een map.

REMOVEFONTS={YES|NO}
Dit kan alle instantie lettertypes verwijderen van de map, diegene die alleen van toepassing zijn op deze map (gespecificeerd door de betreffende object instellingen teksten of in de map instellingen notitieblok). Standaard is NO.
Dit is niet echt een "instelling", maar meer een opdracht dat uitgevoerd wordt op een map.

SHOWALLINTREEVIEW={YES|NO} (alleen &warp4; en hoger)
Laat alle objecten in een open Boomstructuur zien of alleen mappen.

SNAPTOGRID={YES|NO|DEFAULT|EXEC} (&xwp; alleen)
Stelt voor een individuele map in of de "Uitlijn" optie te zien zal zijn in de map zijn &popmenu;. Deze instelling komt overeen met degene op de "Beeld" pagina in de map zijn instellingen notitieblok. YES of NO zal deze instelling voor de map bepalen ongeacht wat er opgegeven is in de globale instellingen, terwijl DEFAULT de instelling de globale waarde zal geven. EXEC zal de "Uitlijn" functie uitvoeren op de map, net alsof u die keuze gemaakt had vanuit de &popmenu;; EXEC is dus geen instelling, maar een opdracht die uitgevoerd kan worden op een map.

SORTCLASS=classname
Dit bepaald het class object om op te sorteren. Het standaard class object is WPFileSystem.

SORTFOLDERSFIRST={YES|NO|DEFAULT} (&xwp; alleen)
Stelt voor een individuele map in of mappen altijd voorop gesorteerd worden of niet. Deze instelling komt overeen met degene op de "Sorteer" pagina in de map zijn instellingen notitieblok. YES of NO zal de instelling voor deze map bepalen ongeacht wat er opgegeven is in de globale instellingen, terwijl DEFAULT de instelling de globale waarde zal geven.

SORTNOW=crit (&xwp; alleen)
Dit zal alle momenteel openstaande overzichten of een individuele map sorteren zonder de map zijn instelling aan te passen (in tegenstelling tot DEFAULTSORT). Dit is niet echt een "instelling", maar meer een opdracht dat uitgevoerd kan worden op een map. crit heeft dezelfde betekenis als met DEFAULTSORT.

STATUSBAR={YES|NO|DEFAULT} (&xwp; alleen)
Stelt de zichtbare status van een individuele status balk in. Deze instelling komt overeen met degene op de "Beeld" pagina in de map zijn instellingen notitieblok. YES of NO zal de instelling van deze map bepalen ongeacht de opgegeven keuze in de globale instellingen, terwijl DEFAULT de instelling de globale waarde zal geven.

TREEFONT=grootte.facename
Instellingen tekst gebruikt om de lettertype aan te passen dat gebruikt wordt voor een Boomstructuur. Bij voorbeeld: 10.Helv

TREESHADOWCOLOR=kleur (alleen &warp4;)
Tekst kleur voor reflectie pictogrammen in de Boomstructuur. kleur mag een standaard kleur naam zijn of een r g b set van waarden (voor rood, groen, blauw waarden).

TREETEXTCOLOR=kleur (alleen &warp4;)
Tekst kleur voor normale pictogrammen in de Boomstructuur. kleur mag een standaard kleur naam zijn of een r g b set van waarden (voor rood, groen, blauw waarden).

TREETEXTVISIBLE={YES|NO} (alleen &warp4;)
De namen van objecten in een map worden wel of niet getoond in een Boomstructuur.

TREEVIEW=stijl1[,stijl2...]
Dit stelt de boomstructuur in op de opgegeven stijlen. Mogelijke waarden zijn: INVISIBLE LINES MINI NOLINES NORMAL
De standaard waarde is LINES, NORMAL
Deze instelling komt overeen met de "Boomstructuur" pagina in een map zijn instellingen notitieblok.

WORKAREA={YES|NO}
Dit stelt de map in als een werkgebied. Standaard is NO.
