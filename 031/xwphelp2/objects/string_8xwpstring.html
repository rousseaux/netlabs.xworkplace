XWPString verzorgt object instellingen reeksen. Ook al is de XWPString class ontworpen om instellingen reeksen op te slaan, u kunt ook instellingen reeksen op een XWPString object zelf toepassen, evenals op elk ander object.

In aanvulling op de instellingen reeksen die ondersteund worden door de WPObject class, ondersteunt de XWPString class de volgende reeksen:

>SETUPSTRING=reeks

De instellingen reeks die in het reeks object opgeslagen moet worden. De reeks mag niet opgebouwd zijn uit karakters die niet toegestaan zijn in Desktop instellingen reeks waarden. De standaard is een lege reeks.

DEFAULTOBJECT={objectid|bestandsnaam|*ingang|NONE}

Zet of verwijdert het standaard object dat opgeslagen is in de reeks van het object. De waarde moet een van de volgende zijn:
  • een object ID, omsloten door kleiner dan/groter dan teken (b.v. DEFAULTOBJECT=<WP_DESKTOP>;);
  • een volledig opgegeven bestandsnaam (b.v. DEFAULTOBJECT=C:\mijnbestanden\iets.txt;);
  • een decimale object ingang, voorafgegaan door een sterretje (b.v. DEFAULTOBJECT=*1432;);
  • de NONE code om het huidige standaard object te verwijderen, als die er is.
De standaard voor een reeks van een object is om geen standaard object te hebben(NONE).

CONFIRMINVOCATION={YES|NO}

Komt overeen met de "Bevestig implementatie" instelling op de "reeks" instellingen notitieblok pagina. De standaard is YES.

NOSTRINGPAGE={YES|NO}

Bij YES, zal de reeks van het object instellingen notitieblok de "reeks" pagina niet tonen. Dit kan handig zijn als u de reeks van objecten in uw eigen programma maakt en u wilt voorkomen dat de gebruiker per ongeluk een zorgvuldig ontworpen instellingen reeks verandert. De standaard is NO.
