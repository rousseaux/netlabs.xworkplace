De "Pictogram" pagina is te vinden in de instellingen notitieblokken van vrijwel elk object en biedt u de mogelijkheid om de object naam, pictogram en andere basis object instellingen aan te passen. Maar voor netwerk objecten kunnen veel van deze instellingen niet aangepast worden, daarom zijn deze velden niet te zien op deze pagina.

&xwp; vervangt de originele "Pictogram" pagina van de Werkplek met een nieuwe, uitgebreidere versie als u deze instelling geselecteerd heeft op de "Kenmerken" pagina in het &xwp;: Instellingen object van &xwp; onder "Algemeen". Deselecteer deze instelling om de originele pagina terug te krijgen.

Druk op "Beginwaarde" om de instellingen te gebruiken die in gebruik waren voor u deze notitieblok pagina opende.

Druk op "Standaard" om de instellingen terug te zetten naar de standaardwaarden, welke normaliter afgeleid worden van de class van het object.

Kijk voor meer informatie in de onderstaande onderwerpen: Let op dat afhankelijk van de object class niet alle velden beschikbaar hoeven te zijn op deze pagina.
