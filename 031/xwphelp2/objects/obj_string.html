Een "Object instellingen reeks" kan iedere instellingen reeks bevatten en als extra een object waarop de instellingen reeks standaard van toepassing is.

Zie: "Instellingen reeksen" voor meer informatie over de WPS instellingen reeksen en een lijst van de meest gebruikte.

U kunt de opgeslagen instellingen reeksen op drie manieren toepassen op objecten:

  • De instellingen reeks die opgeslagen is in de reeks van het object kan toegepast worden op elk ander object door het doel object op de object reeks te laten vallen. Bijvoorbeeld, als u een object reeks heeft dat map instellingen aanpast, laat dan een willekeurig map object op de object reeks vallen, en de map zal conform de wens aangepast worden.

    Denk er in dit geval om, dat het uw verantwoordelijkheid is om er zeker van te zijn dat het doel object de reeks begrijpt. Bijvoorbeeld, als de object reeks map instellingen reeksen bevat, dan heeft het geen nut om die toe te passen op programma objecten.

  • Ten tweede, een object reeks kan een extra "standaard" object bevatten waar de instellingen reeks op toegepast wordt als u dubbelklikt op de object reeks. Dit kan handig zijn voor bepaalde "opdracht" instellingen reeksen die altijd op een bepaald object toegepast moeten worden.

  • Tenslotte, u kunt ook instellingen objecten reeksen toevoegen aan de Configuratie Map van &xwp; om het mogelijk te maken om de instellingen reeks op elke map toe te passen. Zie: Configureren &popmenu;s voor details.
