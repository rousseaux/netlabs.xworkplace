Een instellingen tekst object kan elke object instellingen tekst bevatten en optioneel een object waarop de instellingen string standaard van toepassing is.

Zie "Instellingen teksten" voor meer informatie over de WPS instellingen teksten en een lijst van de meest gebruikten.

U kunt de opgeslagen instellingen teksten op drie manieren toepassen op objecten:

  • De instellingen tekst die opgeslagen is in het tekst object kan toegepast worden op elk object door het doel object op het tekst object te laten vallen. Bijvoorbeeld, als u een tekst object heeft dat map instellingen aanpast, laat dan een willekeurig map object op het tekst object vallen, en de map zal conform de wens aangepast zijn.

    Denk er in dit geval om, dat het uw verantwoordelijkheid is om er zeker van te zijn dat het doel object de tekst begrijpt. Bijvoorbeeld, als het tekst object map instellingen teksten bevat, dan heeft het geen nut op die toe te passen op programma objecten.

  • Ten tweede, kan een tekst object een optioneel "standaard" object bevatten op welk de instellingen tekst toegepast wordt als u dubbelklikt op het tekst object. Dit kan handig zijn voor bepaalde "opdracht" instellingen teksten die altijd op een bepaald object toegepast moeten worden.

  • Tenslotte, u kunt ook instellingen tekst objecten toevoegen in de &xwp; Configuratie Map om het mogelijk te maken om de instellingen tekst op elke map toe te passen. Zie Configureren &popmenu;s voor details.
