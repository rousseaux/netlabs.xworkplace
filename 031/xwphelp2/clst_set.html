De "Class lijst" pagina in het "WPS Class lijst" object maakt een nauwkeurige afregeling mogelijk van de afbeelding van het object.

Indien "Toon alle SOM classes" wordt aangevinkt, zal de class lijst niet alleen de reguliere WPS classes bevatten, maar alle SOM classes die onlangs zijn geregistreerd door het Workplace Shell proces. Dit toont veel meer classes en het zal meer tijd vragen om ze af te beelden. Behalve de SOMObject, SOMClass, SOMClassMgr, Contained,en Container, zijn al deze extra classes niet gedocumenteerd.

Dit is waarschijnlijk alleen nuttig voor nieuwsgierige SOM programmeurs. U zult het "WPS Class Lijst" object moeten sluiten en heropenen om de wijziging te zien.

Als "Toon methode informatie" is geactiveerd, ziet u drie vensters in de geopende "WPS Class Lijst". Het derde venster toont al de omgezette methoden voor een geselecteerde class. Als dit niet is geactiveerd, ziet u slechts 2 vensters. Deze wijziging is direct van kracht.
