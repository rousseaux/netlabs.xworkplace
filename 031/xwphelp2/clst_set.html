De "Class list" pagina in het "WPS Class Lijst" object maakt het u mogelijk tot nauwkeurige afstemming van de afbeelding van het object.

Indien "Toon alle SOM classes" wordt gecontroleerd, zal de class lijst niet alleen de reguliere WPS classes bevatten, maar alle SOM classes die onlangs zijn geregistreerd door het Workplace Shell proces. Dit toont veel meer classes en zal meer tijd vergen om af te beelden. Behalve voor SOMObject, SOMClass, SOMClassMgr, Contained,en Container, zijn al deze extra classes niet gedocumenteerd.

Dit is waarschijnlijk alleen nuttig voor nieuwsgierige Som programmeurs. U zult het "WPS Class Lijst" object moeten sluiten en heropenen om de wijziging te zien.

Als " Toon methode informatie" is geautoriseerd, ziet u drie vensters in de geopende "WPS Class Lijst". Het derde venster toont al de opgeloste methoden voor een geselecteerde class. Als dit niet is geautoriseerd, ziet u slechts 2 vensters. Deze wijziging is direct van kracht.
