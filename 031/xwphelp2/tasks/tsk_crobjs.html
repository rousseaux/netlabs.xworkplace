&xwp; biedt meer mogelijkheden voor u om nieuwe objecten te maken. Naast anderen bestaan de volgende opties:
  • Open de &link_tempsfdr; en sleep een nieuw object af van een model.

  • Om een nieuw object te maken van het zelfde type als een bestaand object, open het &popmenu; van dat object en maak een keuze uit het "Cre‰er een ander" menu.

  • Als u er achter komt dat u vaak een object probeert te maken van het zelfde type, dan vindt u wellicht de "Cre‰er nieuw" menu handiger. Dit menu bevat altijd dezelfde lijst modellen, die gebaseerd is op de &xwp; &cfgfdr;. Standaard staan veel gebruikte object types, zoals mappen, in dat menu, maar u kunt die lijst zelf aanpassen.

    Zie "Aanmaken van objecten met Cre‰er Nieuw" voor meer informatie.
