De &xcenter; "object knop" widget toont een knop die een Werkplek object vertegenwoordigt.
  • Als dat Werkplek object een map is, kunt u op de knop klikken en de map inhoudmenu's bekijken.

  • Als dat Werkplek object geen map is, kunt u het object openen in zijn standaard weergave door op de knop te klikken.
Object knop widgets kunnen gemakkelijk op elke plaats in het &xcenter; toegevoegd worden door het slepen en loslaten van een willekeurig Werkplek object naar een leeg gebied in de &xcenter; balk.
