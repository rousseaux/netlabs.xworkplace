Dit bepaalt of &xwp; u wel of niet toestaat de parameter lijst te wijzigen van een programma object dat gestart is vanuit de &cfgfdr;.
  • Als deze optie gekozen is, zal &xwp; het normaliter toestaan om de parameter lijst van een programma object dat zich bevindt in de &cfgfdr; aan te passen. Dit heeft betrekking op twee mogelijkheden:

    Ten eerste, &xwp; kan de mapnaam aan de parameter lijst toevoegen, tenzij het laatste karakter in de parameter lijst een procent teken is ("%"). Deze mogelijkheid staat Netscape bijvoorbeeld toe om op te starten vanuit de huidige directory.

    Ten tweede, &xwp; kan een "%**C" toets in de parameterlijst vervangen met de huidige inhoud van het klembord.

  • Als deze optie niet gekozen is, zal &xwp; helemaal niet met de parameter lijst gaan rommelen. Op deze manier kunt u op een algemeen niveau het toevoegen van mapnamen uitschakelen zonder "%" tekens toe te voegen aan losstaande programma objecten.
Zie de xWorkPlace User Guide and Reference voor details.
