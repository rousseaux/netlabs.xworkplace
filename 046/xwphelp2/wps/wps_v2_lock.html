Uncheck this box to remove the &os2; default "Lock in place" &popmenu; entry, including the two submenu items.

This setting affects all objects, including folders.

This menu item exists on &warp4; and above only.

If this setting is still enabled, you can select the "No submenu" setting in addition. If this is also enabled, &xwp; will remove the submenu with the "Yes/No" entries and replace it with a single menu item with a checkmark if the "lock in place" setting is enabled for the object.
